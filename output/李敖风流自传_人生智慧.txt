【FL001】破格也是一种生命力
来源章节：001.《李敖风流自传》破题.txt / 《李敖风流自传》破题
主题标签：写作方法;晚年心态;自我叙事
到八十岁还肯用四十天重新组织一生，说明写作和做人都不必被既定格式收编。真正有生命力的人，不是把过去整理成标准履历，而是在暮年仍敢改变叙述方式，让记忆继续流动。

【FL002】跳跃也能成为结构
来源章节：002.《李敖风流自传》凡例.txt / 《李敖风流自传》凡例
主题标签：写作方法;人生回顾;结构感
人生本来不是一条笔直的线，回忆也不必假装整齐。能把片段、重复、旁枝和突转安排成自己的节奏，反而更接近真实经验；所谓结构，不只是目录清楚，也可以是精神上的贯通。

【FL003】真自传要容得下唠叨
来源章节：003.两位老太太.txt / 两位老太太
主题标签：真实;细节;自我叙事
过分精炼的回忆，常把人生磨成样板；看似唠叨的细节，反而保存了一个人的口气、偏爱和失误。做人也一样，不要急着把自己修饰得无懈可击，真实往往藏在那些不够漂亮的枝节里。

【FL004】用世界坐标看个人苦难
来源章节：004.时——一九三五.txt / 时——一九三五
主题标签：时代感;历史意识;比例感
理解自己的时代，不能只把本民族、本地方的伤痛放大到唯一。把个人记忆放进更宽的世界坐标里，既不会减轻苦难的重量，也能避免狭窄的自怜，让判断多一点比例感。

【FL005】身份要经得起事实检验
来源章节：005.地——我住在中国.txt / 地——我住在中国
主题标签：身份认同;证据;自我定位
人可以有强烈的身份认同，但认同不能只靠情绪和口号。真正站得住的身份感，要能拿出来源、经历和证据；一个人越是在大问题上敢讲事实，越不容易被别人替他定义。

【FL006】一生的成果可以是示范
来源章节：006.人——曾经走过这样一位血肉之躯.txt / 人——曾经走过这样一位血肉之躯
主题标签：人生意义;示范;长期影响
有些人生并不以胜利收场，却仍能留下示范。人不一定能改变整个时代，但可以用自己的选择证明另一种活法存在；播种者的价值，不在于亲眼看见丰收，而在于让后来者知道土地曾被翻动过。

【FL007】尊严从不替强者自我审查开始
来源章节：007.鬼——日本鬼子.txt / 鬼——日本鬼子
主题标签：尊严;表达;独立
面对强势叙事时，最先丢掉的往往不是自由，而是自己替自己删减的勇气。一个人要保住尊严，先要停止在心里预先替权力让路；话未必都要说尽，但不能把畏惧装成成熟。

【FL008】本能也要加工成风格
来源章节：008.原来我就是上帝.txt / 原来我就是上帝
主题标签：欲望;创造;自我调度
人的欲望若只停在本能层面，就容易变成粗糙的消耗；经过想象、文字和自知，它才可能转化为风格。人生智慧不是否认欲望，而是让欲望受创造力调度，而不是反过来支配整个人。

【FL009】不要被升级欲望绑架
来源章节：011.声——日本女人蝴蝶夫人.txt / 声——日本女人蝴蝶夫人
主题标签：节制;欲望管理;幸福观
“已经很好”常被人忽略，因为人总想追逐下一个更刺激的版本。懂生活的人要能在 N 已足够时停下来，不把幸福全部押在 N+1 上；否则所谓进步，只会变成无止境的不满足。

【FL010】回忆不是伤逝
来源章节：014.书房之被.txt / 书房之被
主题标签：回忆;时间观;情绪管理
回忆若只用来哀悼失去，就会把过去变成负担；若把它看成经验的延伸，过去仍能继续滋养现在。真正会回忆的人，不是沉溺旧日，而是把旧日重新转化为理解人生的材料。

【FL011】创作可以组合不可能
来源章节：015.《虚拟的十七岁》.txt / 《虚拟的十七岁》
主题标签：创作;想象力;问题意识
小说和想象的价值，正在于把现实中无法同时成立的东西放到一起检验。创作者若只复制现实，作品就少了发问能力；敢构造不可能的处境，才可能逼出真实人生里被遮住的问题。

【FL012】勇气是管理不可测
来源章节：017.愿——我愿我是驯兽师.txt / 愿——我愿我是驯兽师
主题标签：勇气;风险;分寸
驯兽的吸引力不在于打倒猛兽，而在于与危险共处并维持秩序。人生里的勇气也不是鲁莽冲撞，而是在不可预测的人和局势面前，仍能保持判断、距离和动作的分寸。

【FL013】理解自然不取消悲悯
来源章节：019.狮妈妈外出时.txt / 狮妈妈外出时
主题标签：悲悯;自然观;成熟
看见自然界的残酷，并不等于人也必须冷酷。成熟的眼光可以同时承认生存法则和怜悯之心：知道世界并不温柔，所以更要小心保留自己能温柔的部分。

【FL014】读书要防止读成死书
来源章节：023.反对读死书.txt / 反对读死书
主题标签：读书;判断力;实践
书若只增加记忆，不增加判断，就会把人读僵。真正的读书，是把知识带回现实、带回问题、带回自己的选择里；读活书的人会被书打开，读死书的人只会被书压住。

【FL015】成就也需要自我叙述
来源章节：024.自吹自擂，不可免也.txt / 自吹自擂，不可免也
主题标签：自我表达;解释权;成就观
只等别人替你说明，常常等来的不是理解，而是误解或沉默。适度自述不是浅薄自夸，而是夺回解释权；有实力的人要学会把自己的来路、标准和贡献讲清楚。

【FL016】家族记忆要连上时代

家里的遭遇若只当作私事，很快就会被琐碎淹没；把它放回时代里，才看得出普通人如何被历史推挤。真正有重量的家族记忆，是能从一家人的命运照见一代人的处境。

【FL017】回乡是一次历史核对

回到故地，不只是找乡愁，也是在核对记忆和现实之间的差距。成熟的人不会把故乡完全浪漫化，而会承认它有亲切、有残缺，也有被时代改写后的陌生。

【FL018】孤独中要能自己发笑
来源章节：075.“孤笑”.txt / “孤笑”
主题标签：孤独;自处;精神空间
一个人若只能靠热闹维持存在感，就很容易被环境牵着走。能在孤独里和自己谈话、甚至自己发笑，说明内心还有可供周旋的空间；这种空间，是抵抗寂寞和压迫的基本装备。

【FL019】人格不靠名位撑大
来源章节：084.三不朽中的我.txt / 三不朽中的我

名位可以放大一个人的声音，却不能替他制造人格。真正的大人格，来自见识、胆量、恩怨分明和长期一致的选择；外在头衔只是外壳，撑不起里面空的人。

【FL020】报复也可转化为作品

主题标签：报复;写作;行动
受伤以后只会发泄，力量很快就散掉；把报复心转成文字、证据和行动，才可能让伤害者付出持久代价。人生里有些愤怒不能压灭，但必须加工成有方向的作品。

【FL021】为真理可以暂失名誉

【FL023】视野别被一岛一国困住

【FL027】真逸士不买权贵账

【FL028】逃难后先让日常运转

大时代把人抛到陌生地方以后，最要紧的不是立刻谈宏大理想，而是让吃饭、上学、住处和秩序重新接上。能把破碎生活重新运转起来，是人在乱世里的第一种韧性。

【FL030】错过不必永远遗憾
