【DSZN001】大工程先画对照表
来源章节：001.四部备要暨四部丛刊书目対照表例.txt / 四部备要暨四部丛刊书目対照表例
主题标签：规划;方法;知识工程
李敖十七岁做《四部备要》和《四部丛刊》的书目对照，不是为了炫示记忆，而是在大规模知识面前先建立地图。人生做大事也该如此：先看清已有、缺漏、重叠和优先顺序，再谈投入。没有对照表就冲进去，热情很容易被庞杂吞掉。

【DSZN002】资源薄弱时，先补关键缺口
来源章节：001.四部备要暨四部丛刊书目対照表例.txt / 四部备要暨四部丛刊书目対照表例
主题标签：取舍;资源;长期积累
李敖在表例里说经济力薄弱时，不必幻想一口气买齐大套书，而要先买两种丛书都未收的书，年深月久自然有所成。资源不足不是放弃的理由，而是训练取舍的机会；先补不可替代的缺口，胜过把钱力平均撒在好看却重复的地方。

【DSZN003】早年的硬功夫会变成后来的底气
来源章节：001.四部备要暨四部丛刊书目対照表例.txt / 四部备要暨四部丛刊书目対照表例
主题标签：早年训练;积累;自信
李敖回看十七岁坐在图书馆长凳上连日抄录、比对书目，认为后来研究中国文化的成绩并非凭空冒出。年轻时做过的笨重基础工，常在多年后变成判断力和自信的来源；不要小看没人鼓掌的早期训练，它可能正在替未来铺底。

【DSZN004】知识再大，也要先找入口
来源章节：003.介绍世界最大的百科全书.txt / 介绍世界最大的百科全书
主题标签：读书方法;索引;效率
李敖介绍《古今图书集成》时，最强调的不是它有多大，而是索引、目录和分类如何让人进得去。人生面对庞杂问题，先找入口比硬啃更重要；入口不是偷懒，而是把力量放到可行动的位置。不会找钥匙的人，再勤奋也可能只是在门外绕圈。

【DSZN005】分类是让材料听命于问题
来源章节：003.介绍世界最大的百科全书.txt / 介绍世界最大的百科全书
主题标签：分类;问题意识;资料整理
《古今图书集成》把浩瀚古籍按汇考、总论、列传、艺文、纪事等方式重排，价值正在于让材料围绕问题被调动。材料多不等于见识多，若没有结构，丰富只是负担。会分类的人，不是爱整理，而是知道问题一来，资料该从哪里站出来。

【DSZN006】勤奋要配工具，别膜拜低效
来源章节：003.介绍世界最大的百科全书.txt / 介绍世界最大的百科全书
主题标签：工具意识;勤奋;效率
李敖批评旧式读书人不会编索引、轻视类书，只好死记硬翻，各人重复下同样的笨功夫。勤奋当然可贵，但勤奋若拒绝工具，就会变成低效的自我感动。真正聪明的努力，是让方法帮助体力，而不是把辛苦本身当成美德。

【DSZN007】好索引是在替后来者修路
来源章节：002.《古今图书集成》研究.txt / 文星版的重要
主题标签：索引;公共贡献;后来者
文星版《古今图书集成》最划时代的贡献，被李敖放在索引上：它不只方便自己查书，还校正前人、便利今人与后人。真正有公共性的工作，常不是一个人找到答案就结束，而是把通道修好，让后来者少走冤路。

【DSZN008】藏有大书不如真会使用
来源章节：002.《古今图书集成》研究.txt / 文星版的重要
主题标签：使用;工具书;实用
李敖引《品花宝鉴》里把《古今图书集成》说成摆设的故事，正好提醒人：拥有工具和使用工具是两回事。书架、设备、头衔都可能只是装饰，真正的价值要看它们能否被调动来解决问题；不会用，再贵重也只是占地方。

【DSZN009】大作品背后要追真正出力的人
来源章节：002.《古今图书集成》研究.txt / 成书的疑案
主题标签：功劳;实际劳动;公道
《古今图书集成》长期被归到皇帝和奉命大臣名下，李敖却一层层追出陈梦雷的实际编纂之功。看任何成果，都要问谁真正做了长期、核心、不可替代的劳动。名义上的光环越耀眼，越不能让真正扛活的人消失在阴影里。

【DSZN010】官方叙事越整齐，越要查证据链
来源章节：002.《古今图书集成》研究.txt / 成书的疑案
主题标签：权威叙事;证据链;考证
雍正的序文和诏令把成书故事说得冠冕堂皇，李敖却从陈梦雷自述、版本事实、后人考证里找裂缝。整齐叙事最容易让人放松警觉；判断重大功过时，不能只看权威怎样署名，还要查材料之间能不能互相咬合。

【DSZN011】版本和物证会替真相说话
来源章节：002.《古今图书集成》研究.txt / 版本的种类
主题标签：版本;物证;真相
李敖辨析《古今图书集成》的铜活字、铅字本、石印本、影印本，不是嗜古癖，而是用版本痕迹判断事实。很多真相不只藏在人的说法里，也藏在纸张、版式、流传和缺册中。会看物证，才不容易被漂亮口供牵着走。

【DSZN012】长期积累，才撑得起一场大成
来源章节：002.《古今图书集成》研究.txt / 成书的疑案
主题标签：积累;大工程;沉潜
陈梦雷能在四年半间编成巨帙，背后是五十年读书、一万五千余卷材料和长期训练。人生里许多所谓爆发，其实只是积累终于有了出口；没有前面的沉潜，临时用力再猛，也很难做出真正经得起时间的大工程。

【DSZN013】让稀有知识变得可查可用，是功德
来源章节：002.《古今图书集成》研究.txt / 文星版的重要
主题标签：知识公共化;出版;可及性
李敖写《古今图书集成》一再说明它稀有、昂贵、难得，文星版的意义就在于缩小册数、降低门槛、加索引、便检索。知识若只作为少数人的秘藏，公共价值就被锁住；把珍贵材料变得可接近、可查、可用，本身就是一种功德。

【DSZN014】艰深知识要有通往民间的路
来源章节：004.中国历史演义总说.txt / 中国历史演义总说
主题标签：通俗化;传播;公共知识
李敖谈历史演义，是因为正史太深、太杂、太片断，普通人难以进入。严肃知识若永远只在专家屋里打转，就很难成为社会的共同记忆。把深东西讲清、讲活、讲到人愿意听，不是降低学问，而是让学问有路可走。

【DSZN015】故事能教育，也能改写记忆
来源章节：004.中国历史演义总说.txt / 关于《三国演义》
主题标签：故事;事实;大众记忆
李敖指出《三国演义》给中国人带来文字、思想和民间教育，也把许多并非史实的情节变成大众记忆。故事的力量正在这里：它能把人带进知识，也能替事实化妆。人生中爱听故事可以，但不能忘了故事不是事实本身。

【DSZN016】会读通俗书，才得其利不受其害
来源章节：004.中国历史演义总说.txt / 关于《三国演义》
主题标签：阅读方法;通俗书;复按
李敖说读《三国演义》要会读，会读便得其利、不见其害。通俗书最适合作入口，却最怕读者把入口当终点。成熟的读法，是先借它建立兴趣和脉络，再回到史料复按；既不装高雅地拒绝通俗，也不被通俗牵着鼻子走。

【DSZN017】好观念也要看时代土壤
来源章节：004.中国历史演义总说.txt / 关于《后汉演义》
主题标签：改革;时机;现实条件
李敖写王莽，说他有些改革带着人道意味，却忽略传统保守势力和自身局限，终于生不逢时。好意和先进观念不是成功保证；做事要看社会承受力、执行条件和人的可信度。把正确想法放错土壤，也可能长成灾难。

【DSZN018】真正自信的掌权者容得下不合作
来源章节：004.中国历史演义总说.txt / 关于《后汉演义》
主题标签：权力;独立人格;宽容
光武帝推崇不与王莽合作的人，也尊重不跟自己合作的严光，这在李敖眼里显示了高水准。权力最容易把顺从误认成德行，把拒绝当成冒犯；真正有自信的人，能分清独立人格和敌意，不靠把所有人纳入麾下来证明自己。

【DSZN019】一姓一族的私心会毁公共局面
来源章节：004.中国历史演义总说.txt / 关于《两晋演义》
主题标签：私心;公共性;制度后果
李敖谈晋朝，抓住司马氏疑天下、信同姓，最后八王之乱与五胡乱华相接。公共事务一旦被家族私心接管，制度就会为少数人的安全感付出巨大代价。越是掌握大权，越不能把天下当成自家防盗门。

【DSZN020】开放要配实力与边界
来源章节：004.中国历史演义总说.txt / 关于《唐朝演义》
主题标签：开放;实力;边界
李敖称唐朝文化上能大力胡化，却在武力上未能力大化胡，形成文武分离。开放本身值得珍惜，但开放不是没有边界和实力的天真；一个人或组织吸收外来养分时，也要有判断、制度和自保能力，才不把胸襟变成漏洞。

【DSZN021】看见敌文仍惜才，才是真人才观
来源章节：004.中国历史演义总说.txt / 关于《唐朝演义》
主题标签：人才;胸襟;用人
武则天读骆宾王讨伐自己的文章，不只不以被骂为怒，反而责怪宰相没有争取人才。许多人说爱才，只爱替自己说好话的人；真正的人才观，是在刺耳、反对甚至攻击里仍能看见本领。能把面子放下，才可能把人才留下。

【DSZN022】读乱世先问百姓付了什么账
来源章节：004.中国历史演义总说.txt / 关于《五代演义》
主题标签：百姓;代价;历史眼光
李敖写五代十国，不只数朝代更迭，更反复指出决黄河、打草谷、人口锐减中受害最深的是老百姓。看历史和看现实一样，不能只盯着胜负者的名字；真正的代价常由无名者支付。若不问谁承受，所谓大局就会很冷。

【DSZN023】对内横对外软，是虚弱的强势
来源章节：004.中国历史演义总说.txt / 关于《宋朝演义》
主题标签：勇气;强弱;权力
李敖形容宋朝是在羔羊前是勇士、在勇士前是羔羊，对内压低宰相和武人，对外却窝囊退让。人生里也有这种假强势：只敢对弱者摆威风，遇到真正压力便退缩。检验勇气，要看他面对强者时怎样，而不是欺负弱者时多大声。

【DSZN024】不要被名词窄化现实
来源章节：004.中国历史演义总说.txt / 关于《元朝演义》
主题标签：名词;定义;历史视野
李敖说元朝不该只按国号入主中原以后来算，还要把蒙古前期和汗国背景纳入视野。名词为了方便而生，却常把复杂现实切窄。做判断时，要警惕定义替你偷工减料；先问事实边界在哪里，再决定名字怎么用。

【DSZN025】别让后朝宣传替你写前朝
来源章节：004.中国历史演义总说.txt / 关于《元朝演义》
主题标签：宣传;历史判断;胜利者叙事
李敖指出明朝继元之后，容易用民族大义和夷夏之防曲解元朝，甚至有人直接把元朝出局。后来者掌握话筒，不等于掌握真相。看任何前任、败者或被取代者，都要把继任者的宣传折扣算进去，免得把清算当历史。

【DSZN026】评历史先拆种族偏见
来源章节：004.中国历史演义总说.txt / 关于《元朝演义》、关于《清朝演义》
主题标签：偏见;民族观;历史判断
李敖谈元朝、清朝，都要求取消汉人本位的种族偏见，用更博大的中华民族眼光看问题。偏见最坏之处，是它先替你决定谁该被丑化，再让材料去服从。成熟的判断，要先拆掉身份滤镜，再谈功过是非。

【DSZN027】独夫集权会制造小人当道
来源章节：004.中国历史演义总说.txt / 关于《明朝演义》
主题标签：集权;制度;小人
李敖写明太祖废丞相、用锦衣卫和太监特务，指出这不是普通中央集权，而是皇帝一人集权。制度不给正当分权留下位置，就会让亲信、特务和小人填补空白。一个组织若只剩最高者意志，下面自然长出最会逢迎的人。

【DSZN028】评失败要算时代遇到的新难题
来源章节：004.中国历史演义总说.txt / 关于《清朝演义》
主题标签：时代变局;公平判断;失败
李敖论清朝后期，不只骂它丢人，也提醒它撞上数千年未有之变局、千年来未有之强敌。评人评事不能只看结局难看，还要看对方面对的是旧题还是新局。把前所未有的难题当普通失误审判，判断会显得痛快，却未必公平。

【DSZN029】民族口号要升格为共同体眼光
来源章节：004.中国历史演义总说.txt / 关于《民国演义》
主题标签：共同体;政治成熟;民族观
李敖看重民国成立前后从驱逐鞑虏转向汉满蒙回藏五族一家，认为这是政治上的第一进步。口号在动员时也许有效，但真正建国、共处、解决问题时，必须从排斥语言升格为共同体语言。能及时升级口号，是政治成熟，也是做人处世的成熟。

【DSZN030】熟词也有来路，别把习惯当当然
来源章节：004.中国历史演义总说.txt / 关于《二十五史演义》和《新中国未来记》
主题标签：词源;追问;文化交流
李敖提醒“历史”二字本身也有日本移入、学制改革和文化交流的来龙去脉。越熟悉的词，越容易被人当成天然存在；真正的理解要连日常名词也肯追问来源。习惯不是证据，熟悉也不是终点。

【DSZN031】爱国要从理解开始
来源章节：004.中国历史演义总说.txt / 关于《二十五史演义》和《新中国未来记》
主题标签：爱国;理解;理性
李敖说读完这部大书，可以变成理智的爱国者，因为你真的透过它了解了中国。爱国若只剩情绪，很容易被口号牵走；理解过历史的复杂、光荣、耻辱和困局，才可能爱得清醒。清醒的爱，比热闹的爱更经得起考验。
