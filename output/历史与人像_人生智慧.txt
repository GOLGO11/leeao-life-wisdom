【LSRX001】口头中立，要看实际偏向
来源章节：001.袁世凯的祀孔.txt / 袁世凯的祀孔
主题标签：政治判断;传统;中立
袁世凯一面讲宗教信仰自由，一面又把孔教抬到特别位置，李敖看见的是宣告与实际之间的裂缝。判断一个人或制度是否公平，不能只听他说一视同仁，而要看资源、仪式、法令和庇护实际落到谁身上；嘴上平等而手上加冕，所谓中立只是遮羞。

【LSRX002】看人要看整组人格
来源章节：002.余玠人格品质的分析.txt / 余玠人格品质的分析
主题标签：识人;人格;复杂性
李敖分析余玠，不把他压扁成忠臣、儒生或名将，而是同时看贫寒、野心、才具、暴烈、任事、骄矜和贪财。识人也该如此：一个人的优点和危险常在同一组人格里并存，若只拿一个标签判断，就会既看不见他的可用之处，也看不见他会出事的地方。

【LSRX003】招贤的气度在善待没用上的人

主题标签：用人;礼遇;格局
余玠开招贤馆，不只欢迎来客，连建议不能采用的人也给礼物、致谢、送走。真正会用人的人，懂得人才来投不是一次性交易；即使当下不能用，也要让对方带着尊严离开。善待没用上的人，才看得出一个人的格局和长线眼光。

【LSRX004】用别人的谋略，要把名字还给别人

主题标签：功劳;谋略;公道
冉氏兄弟献策，余玠没有把谋略吞成自己的功劳，而是请求朝廷任用他们。借用别人的主意并不可耻，可耻的是把别人的心血抹掉。会成事的人，要能让功劳回到来源处；这既是公道，也是继续聚拢聪明人的办法。

【LSRX005】真正的负责人先替别人挡失败
来源章节：002.余玠人格品质的分析.txt / 余玠人格品质的分析
主题标签：责任;领导;担当
余玠推动筑城，面对反对者说成则蜀赖其利，不成则他独自承担，别人不受牵连。负责人最有力量的承诺，不是保证必胜，而是把失败责任先揽到自己身上。能替部属和合作者挡住后果，事情才有人敢放手去做。

【LSRX006】功成之后更要管理自己的危险
来源章节：002.余玠人格品质的分析.txt / 余玠人格品质的分析
主题标签：功成;自保;节制
余玠有才有功，也因功成后的骄气、贪财和不懂自保而走向危险。人在低处容易努力，在高处更要收束，因为成绩会带来资源，也会带来嫉恨和权力的猜忌。功成不是松懈的开始，而是更精细地管理锋芒、欲望和处境的开始。

【LSRX007】教育不是预备生活，教育就是生活
来源章节：003.杜威的教育思想及其他.txt / 杜威的教育思想及其他
主题标签：教育;生活;能力
李敖借杜威说明，教育不是把人暂时关在学校里，等将来才开始生活；教育本身就是生活，是人在经验中发展能力、学习与社会相处的过程。把学习和人生切开，容易培养只会应付科目的学生；把学习放回生活，知识才会长成能力。

【LSRX008】看见真实痛苦，就不能再躲进禁忌
来源章节：004.桑格夫人和节育运动.txt / 桑格夫人和节育运动
主题标签：人道;禁忌;改革
桑格夫人真正投身节育运动，是因为亲眼看见贫穷母亲被连续怀孕、非法堕胎和死亡逼到绝路。许多禁忌之所以该被打破，不是为了刺激社会，而是因为它遮住了活人的痛苦。判断一个议题要先问谁在受苦，而不是先问旧礼教准不准谈。

【LSRX009】坐牢不是目的，改变才是目的
来源章节：004.桑格夫人和节育运动.txt / 桑格夫人和节育运动
主题标签：改革;行动;目标
桑格夫人说监狱不是最后目的，她要先做该做的事，再承担跟来的后果。真正的改革者不是把受难当姿态，而是清楚知道自己要改变哪条法律、哪种制度、哪类痛苦。牺牲若不能推动事情前进，就容易只剩悲壮；行动要比姿态更重要。

【LSRX010】运动做大以后，视野也要跟着放大
来源章节：004.桑格夫人和节育运动.txt / 桑格夫人和节育运动
主题标签：改革;视野;成长
桑格夫人早年争的是避孕知识能不能公开，后来把方向扩展到世界人口与家庭计划。一个人或一场运动若有生命力，就不能永远停在最初的战壕里；局面变大，问题变深，视野也要随之扩大。坚持不是重复旧口号，而是把初衷带到更广处。

【LSRX011】正确的事若只能沉默推进，社会还没真正进步
来源章节：004.桑格夫人和节育运动.txt / 桑格夫人和节育运动
主题标签：公共讨论;改革;勇气
蒋梦麟想在台湾推节育，却劝桑格夫人不要来，只能沉默地做。李敖的痛感在于，有益国计民生的事若还怕公开讨论，说明社会的勇气和智慧都在后退。好事可以从谨慎开始，却不能以永远偷偷摸摸为成熟；公共理性终究要走到阳光下。

【LSRX012】好婚姻容得下彼此的名字
来源章节：004.桑格夫人和节育运动.txt / 桑格夫人和节育运动
主题标签：婚姻;尊重;主体
桑格夫人再婚后仍保留原来的名字，丈夫也不以为忤，李敖称这种雅量是佳话。亲密关系不是把一个人并入另一个人的名下，而是在共同生活中保留彼此的志业和身份。爱若必须吞掉对方的名字，常常已经不是爱，而是占有。

【LSRX013】别把法律案件煮成道德公审
来源章节：005.纪翠绫该生在什么时候？.txt / 纪翠绫该生在什么时候？
主题标签：法律;道德审判;事实
纪翠绫案本是法律案件，却被羞耻、五伦、狗男女之类词语煮成社会公审。李敖提醒人，判断事情要先分清法律事实、伤害程度和私人选择，不能让泛道德词汇抢走全部话语权。道德一旦替代事实，旁观者很容易在正义感里参加围猎。

【LSRX014】用历史长度拆掉当下的理直气壮
来源章节：005.纪翠绫该生在什么时候？.txt / 纪翠绫该生在什么时候？
主题标签：历史意识;道德;反思
李敖把历代通奸法律和私刑一层层排出来，是为了让当下那些自以为天经地义的责骂露出历史根脉。人们越理直气壮时，越需要把观念放进长时段里检验；许多今天被说成永恒道德的东西，细看只是旧权力留下的习惯。

【LSRX015】大气压会压倒勇者，不必用屈服否定真情

主题标签：社会压力;真情;同情
李敖说纪翠绫已经有勇气和坦率，即使最后被社会压力逼成痛改前非的姿态，也无损于她的理想和真情。人在巨大的社会大气压下低头，不等于曾经的真实全是假的。判断受压者时，要看她面对压力前说过什么、做过什么，而不是只看最后被压出的姿势。

【LSRX016】解释有空间，证据有边界
来源章节：007.“两昆仑”考.txt / “两昆仑”考
主题标签：考证;解释;证据
“两昆仑”有多种解释，李敖逐一比较梁启超、谭夫人传说、陶菊隐记录等说法。诗句可以有解释空间，但解释空间不等于任意发挥；越是漂亮的说法，越要问证据从哪里来。允许多解，是心胸；守住证据边界，是责任。

【LSRX017】选才不能靠烧参考书

宋代禁科场书，想防止士子剽窃陈言，却挡不住考试参考书继续流通。真正的选才不能只靠查禁资料、焚毁书板来维持纯洁，因为需求仍会从别处冒出来。若考试制度本身诱人背答案，烧掉参考书也只是烧表面，不是改问题。

【LSRX023】懂法要明理，别把条文读成石头
来源章节：010.夫妻同体主义下的宋代婚姻的无效撤销解消及其效力与手续.txt / 夫妻同体主义下的宋代婚姻的无效撤销解消及其效力与手续

李敖引用“泥法而不明于理，不可以言法”的判断，指出法律不能被机械理解。真正懂规则的人，不是背得出条文，而是知道条文为何而设、保护谁、怎样避免制造荒谬。没有道理支撑的依法，常会把法治读成石头。

【LSRX024】替人洗冤要同时守证据和边界

李敖替李清照辨再嫁之诬，一面核史料可靠性、时间线、法律制度和常理，一面又说明再嫁本身并不可耻。真正替人洗冤，不是迎合旧道德证明她纯洁，而是把不实指控拿掉，同时守住个人选择的边界。证据和人道，要一起站稳。

【LSRX025】私事不该被谣言拖上道德刑场

【LSRX027】史料有等级，不能把演义当档案

【LSRX028】存疑不是免考证的许可证
来源章节：012.对《徐树铮先生文集年谱合刊》的批评.txt / 对《徐树铮先生文集年谱合刊》的批评

【LSRX029】私德好，不能抵消公共错误
来源章节：012.对《徐树铮先生文集年谱合刊》的批评.txt / 对《徐树铮先生文集年谱合刊》的批评

李敖批评把曹、章、陆说成私德学问好、未取回扣，就想淡化重大政治责任。一个人有没有私下好处，和他在公共事务中造成什么后果，是两本账。做人不能因为某人温和、清廉、会读书，就忘了追问他的能力替什么方向服务。

【LSRX030】用动机洗白行为，常会越洗越黑
来源章节：012.对《徐树铮先生文集年谱合刊》的批评.txt / 对《徐树铮先生文集年谱合刊》的批评

徐谱替徐树铮辩解，说他内心未必亲日、杀陆也许有苦衷；李敖指出这些动机说法遮不住实际行为。评判人不能只听他心里怎样想、不得已有多少，而要看他做了什么、造成什么后果。动机若被拿来反复洗白行为，常常只是让污点轮廓更清楚。
