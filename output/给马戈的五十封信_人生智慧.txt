给马戈的五十封信 人生智慧

条目数：20

【GMG001】不问收获，也能成全一段美
来源章节：001.无闻之美.txt / 无闻之美
主题标签：得失;洒脱;关系
李敖说有些事结果不明朗，反倒保留无闻之美。人生不是每一份投入都要立刻追问收成；当一段关系、一次帮助或一种心意已经尽力，知道适时收手、不再追索回报，反而能保存它的清洁和余味。

【GMG002】写日期，是替记忆留坐标
来源章节：005.年月日要写清楚.txt / 年月日要写清楚
主题标签：记录;史料;记忆
李敖提醒写信要把年月日写清楚，因为一封信日后可能成为共同生活和思想往来的史料。很多关系消失得不是内容，而是时间坐标；愿意把日期、情境和来龙去脉写清楚，是对自己经验负责。

【GMG003】写东西要舍得用力气
来源章节：006.活页日记.txt / 活页日记
主题标签：写作;认真;表达
他嫌草率操觚不成样子，背后是对文字劳动的尊重。表达不只是把情绪倒出来，还要让别人看得见你的认真、秩序和分寸；写得用力，也是做人不敷衍的一部分。

【GMG004】心软之外，还要练一点狠心肠
来源章节：008.提倡狠心肠.txt / 提倡狠心肠
主题标签：坚强;边界;感情
李敖劝马戈不要勉强多写信，也赞成她在感情之外培养坚强。重感情若没有边界，常会把自己拖进无力；必要时能少写、能拒绝、能把心收回来，才是真正保护长期情义的硬度。

【GMG005】浪漫要有双方承受的空间
来源章节：009.爱情牢骚.txt / 爱情牢骚
主题标签：爱情;浪漫;分寸
恋爱中的浪漫不是单方面越多越好，而要看两个人共同能容许多少。一个人把生活想成诗，另一个人若只觉得负担，浪漫就会变成压力；成熟的亲密关系，要让美感落在双方都承受得住的尺度里。

【GMG006】狷介也要练习与人相处
来源章节：015.小舟之逝，从此始矣！.txt / 小舟之逝，从此始矣！
主题标签：交游;原则;处世
李敖劝人多交游、善与人交。特立独行若只剩拒人千里，容易把原则变成孤僻；能在不出卖自己的前提下与人周旋，才算把锋芒磨成可用力量。

【GMG007】得意失意都不要改本色
来源章节：015.小舟之逝，从此始矣！.txt / 小舟之逝，从此始矣！
主题标签：本色;人格;境遇
李敖主张无论腾达还是落魄，都正大光明地显示自己，不藏好也不掩真。人的可靠处，不在顺境时多漂亮，而在境遇改变后仍能认出同一个核心；不改本色，是给自己留下连续的人格。

【GMG008】给自己开一张取舍单
来源章节：015.小舟之逝，从此始矣！.txt / 小舟之逝，从此始矣！
主题标签：取舍;标准;原则
他建议先规列做人的尺度和标准，合则留，不合则去。人生的许多狼狈来自临场才问该不该，结果被情面、欲望和压力推着走；提前写清楚哪些可取、哪些不可取，关键时刻才有东西替你站住。

【GMG009】长大不是悲哀，是不逃避现实
来源章节：018.譬如赤子.txt / 譬如赤子
主题标签：成长;现实;成熟
李敖反对把成长看成纯粹失落，认为人应当高兴自己正在长大。成熟不是把天真全部埋掉，而是承认现实已经来到眼前；愿意长大的人，才有能力把纯真变成判断，而不是把它留成逃避。

【GMG010】赤子之心要留下求知和坦白
来源章节：018.譬如赤子.txt / 譬如赤子
主题标签：赤子之心;求知;坦白
他理解的赤子之心，不只是幼稚可爱，而是坦白、好奇、求知、不玩手段。真正值得保留的童心，不是拒绝复杂世界，而是在复杂中仍保有提问、模仿、直觉和诚实感受的能力。

【GMG011】和光同尘不是屈服，是少惹无谓麻烦
来源章节：021.只消捡起一块石头.txt / 只消捡起一块石头
主题标签：处世;藏锋;原则
李敖把和光同尘看成一种减少麻烦的方法，而不是向庸俗投降。一个人可以在外形上柔一点、在日常中不处处硬碰，同时把核心原则握住；会藏锋，不等于没锋。

【GMG012】开一次花，也可以开得珍贵
来源章节：023.寡情如此.txt / 寡情如此
主题标签：价值;节制;独特
他从昙花想到一种不轻易、不贪多的生命姿态。人生不必把所有能力都摊开成热闹，也不必证明自己时时盛放；有些价值恰在少、精、准，让人记住一回就够。

【GMG013】先减少刺激，再谈抵抗刺激
来源章节：027.为前信进一解.txt / 为前信进一解
主题标签：自制;刺激;策略
李敖说点火以后再灭，不如一开始不点；善战者不战，坚强者不轻用坚强。自制不是天天把自己放进诱惑里硬扛，而是懂得撤掉不必要的刺激源；少打无谓战，反而保存了真正要用的力量。

【GMG014】感情要相称，别把报答做成失衡
来源章节：043.非“痴”.txt / 非“痴”
主题标签：感情;分寸;关系
他借豫让故事指出，人以国士待我，才以国士报之；感情太多或太少都失了分寸。人与人相处，不是越热烈越高尚，而是要看对方给予什么、理解什么、承担什么；报以相称，才不把情义变成自我表演。

【GMG015】别用唯美心理曲解现实
来源章节：045.非“痴”又一章.txt / 非“痴”又一章
主题标签：现实;自欺;成熟
他批评鸵鸟式逃避现实，以及用曲解事实满足唯美心理。美感可以安慰人，却不能替事实作证；真正成熟的眼光，是在现实不美时仍看清现实，而不是把它改写成自己愿意相信的样子。

【GMG016】短痛有时胜过长期钝痛
来源章节：045.非“痴”又一章.txt / 非“痴”又一章
主题标签：决断;清醒;痛苦
李敖说与其长期头痛，不如干脆昏倒一次，意思是有些人需要一次震荡才会清醒。人生中拖着不决最耗人，短时间的痛苦整理、分离或翻案，可能换来彻底醒悟；怕痛而一直拖，反而把痛拉长。

【GMG017】心要有一个感兴趣的问题安顿
来源章节：045.非“痴”又一章.txt / 非“痴”又一章
主题标签：兴趣;研究;安顿
他认为人生快乐条件之一，是时时有个自己感兴趣的问题可研究。人若只被情绪和人事牵着走，心很容易无处落脚；给自己一个真正愿意追问的问题，等于给生活装上一根稳定的轴。

【GMG018】最后一次必须真有边界
来源章节：049.好多个“最后一次”.txt / 好多个“最后一次”
主题标签：边界;决断;拖延
他讽刺一个接一个的最后一次其实不像结束。人说要停止、告别、收手，最要紧的是给自己设下可验证的边界；若每次破例都算最后一次，所谓决断就会沦为拖延的装饰。

【GMG019】友情疏淡，有时只是方向不同
来源章节：051.跋《给马戈的五十封信》.txt / 跋《给马戈的五十封信》
主题标签：友情;变化;成熟
李敖回看朋友逐渐疏淡，认为个人演化方向不同不足为奇。关系变远不必都解释成背叛或亏欠；人各自成长、困顿、转弯，能承认差异，也就少一点把旧情强行拉回原状的怨气。

【GMG020】人生方式要向未来兑现
来源章节：051.跋《给马戈的五十封信》.txt / 跋《给马戈的五十封信》
主题标签：志向;兑现;人生方式
李敖说十年来把想象中十年后的自己一一挤兑，像个立小志的完成家。真正的志向不只在豪言，而在几年以后能不能逐项兑现；把未来的自己当成债主，日常选择就不容易虚浮。
