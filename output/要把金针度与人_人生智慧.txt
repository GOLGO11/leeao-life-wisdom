李敖比较杜甫读万卷书与陈梦雷编《古今图书集成》，看重的是后者把私人读书积累做成后人可用的大型工具。学问若只停在个人胸中，仍是私藏；经过编排、索引和整理，能让后来者少摸黑，才成为公共财富。人生里的能力也是如此，最高用法不是自夸拥有多少，而是把复杂东西做成别人也能使用的阶梯。

【YBJZ002】旧东西要活用，才不是负担

李敖说古书本是朽腐，除非能化朽腐为神奇，能利用、活用，否则对现代人没什么用处。人生面对传统、经验和旧材料，也不能只会供奉；旧东西要被重新照亮，能解决眼前问题，才不是背在身上的沉重包袱。

【YBJZ003】读书得间，是读出缝里的学问

李敖所谓“读书得间”，不是把字面背熟，而是能读出书里的窍，读出字里行间没直接说出的关系。做人做事也一样：只看表层信息，常会被文本、场面和口号牵走；能在缝隙里看见结构和动机，判断才会变深。

【YBJZ004】先分清理想、规范和事实

李敖提醒读古书要分清书生理想、道德法律和社会事实，否则会把托古改制当成真实历史，把规范口号当成社会现象。人生判断也要这样分层：愿望、规定、事实各有位置，混在一起，就会把漂亮话误认成真实世界。

【YBJZ005】方法错了，苦读也会成书呆子

李敖讽刺许多人白首穷经，最后只是有脚书橱，因为方法不当，材料看过就丢掉。努力当然重要，但错误方法会把努力变成疲劳。人生里许多失败不是不勤奋，而是勤奋没有结构；苦读苦干之前，先检查自己是不是在用笨办法磨损生命。

【YBJZ006】工具书要找得到，也要看得懂

李敖判断工具书有两个标准：容易找到需要的答案，也容易了解找到的答案。工具不是越大越好、越有名越好，而是要把人带到答案前。人生里的工具、制度和流程也如此：若让使用者头痛，名气再大，也没有真正服务问题。

李敖引夏曾佑、陈寅恪说中国书虽多，基本不过几十种，正是要替读者从浩瀚中定出门径。面对巨大知识，不必幻想把海喝干；先抓真正能支撑全局的基本书、基本问题和基本方法，反而能更快进入核心。

李敖欣赏闻一多说自己不是蠹虫，而是杀蠹的芸香：越恨故纸堆里的腐朽，越要把它弄明白。真正的批判不是远远骂几句，而是深入到对象内部，拆出它的结构和病灶。懂得越深，反抗才越有准头。

李敖自称几十年天天读书，但真正自豪的是读得多而不成书呆子，能保持博学、开明和新鲜。长期学习容易让人变硬、变旧、变自满；好方法的价值，正在让知识不断更新人，而不是把人封进旧书壳里。

【YBJZ012】跳读不是浮滑，是抓重点

李敖反对逐字逐句死读，主张让每页重点跳出来。跳读不是敷衍，而是训练眼睛迅速识别有价值的信息；非重点十行二十行可以略过，重点半行也要扣住。人生处理信息，也要分清什么值得慢，什么应该快。

【YBJZ013】眼到手到，材料才归你所用

李敖读书时随看随动手，勾、剪、割、影印，再分类处理，让一本书永远为自己所用。知识只在眼前滑过，很快会散；能被标记、摘出、归档、复用，才真正进入你的系统。读书如此，做任何资料工作也如此。

【YBJZ014】让一个问题牵出一排书

李敖说看到重点有兴趣，就同步找来许多相关书钻研，名为“獭祭法”。真正的学习不是一本书孤零零地读完，而是让一个问题牵动一组材料，彼此印证、补充、拆台。问题活了，书才会互相说话。

【YBJZ015】时间要挑剔，别喂给烂书

李敖说哪本书值得跳看，他挑剔得很；许多满纸废话的书，除非做反面教材，否则不看。人生时间有限，选择比忍耐更重要。不是每个材料都配消耗你，学会闻出坏鸡蛋，是保护精神和时间的基本能力。

【YBJZ016】保护清单也可能变成毁灭路线

李敖写乾隆编《四库全书》时一面征书编书，一面禁毁书籍，提醒人们目录、清单和征集并不天然等于保存；在权力手里，它们也可能变成查禁和销毁的路线图。做整理工作要警惕这个反面教训：把材料暴露出来以后，还要看谁掌握处置权。没有自由和善意，保存的名义也可能帮助毁灭。

【YBJZ017】大而不用，价值也会被锁住

李敖说《四库全书》虽是最大丛书，却因只有七部，流传不广，所以只见其大，不得其用。人生做工程、建系统、积累资源，不只要追求规模，也要追求可及性。不能被用起来的巨大，很容易只是壮观的锁柜。

【YBJZ018】著述贵博，更贵精

李敖赞胡应麟主张学问又精又博，既不以鸿硕自居，也不以空疏自废。博而不精会散，精而不博会窄；真正可靠的能力，是广见材料后仍能抓准问题，细看一处时又不失全局。

【YBJZ019】史学不能只堆材料

李敖写章学诚超出考据圈，指出单纯编纂和考证不算史学，史学还得有个人见解。做任何研究都如此：材料是基础，却不是结论。只堆资料而没有问题意识和判断，最多是仓库，不是思想。

【YBJZ020】每年有定稿，才见真用功

【YBJZ022】承认别人更强，是服善之道

李敖称赞张载公开告诉学生，二程深明易道，自己不及，让学生可以师事他们。这种承认别人胜过自己的气度，比争名更难。真正爱真理的人，不怕学生看见别人的长处；服善，才是学问人格的一部分。

【YBJZ023】学问要明道救世，不要清谈空言

主题标签：实学;现实;责任
李敖写顾炎武痛恨知识分子逃避现实，认为学问是为了明道救世，不是徒作诗文清谈。读书若只让人会说漂亮话，却不让人看见世道、承担问题，就会变成空转。真正的学问，最后要回到现实中的人和事。

【YBJZ024】知一真知，胜过知十假知
来源章节：004.哲学类.txt / 哲学类 / 戴震：《戴氏遗书》
主题标签：真知;求实;学习
李敖引戴震说，知道十件都不真，不如知道一件真知。知识数量容易诱人自满，真伪却决定它能否立住。人生求知不必急着堆很多看似懂的东西，先把一个问题弄到可靠，比十个模糊结论更有力量。

【YBJZ025】以理杀人，也要警惕
来源章节：004.哲学类.txt / 哲学类 / 戴震：《戴氏遗书》
主题标签：伦理;人情;警惕
李敖看重戴震反对不近人情的理学，指出儒者以理杀人和酷吏以法杀人并无二致。最可怕的伤害，常披着高尚名义出现。做人讲原则不能脱离人情与事实；若把抽象道理压到人身上，道理也会变成刑具。

【YBJZ026】异端相观，才能照见己偏
来源章节：004.哲学类.txt / 哲学类 / 焦循：《焦氏丛书》
主题标签：宽容;异端;自我修正
李敖写焦循对异端态度开明，主张以彼之善观我之不善，也以我之善摩彼之不善。真正的自信不是不能容人，而是敢让不同观点照出自己的偏。把异端一概驱逐，失去的不只是对方，也包括修正自己的机会。

【YBJZ027】先知也会被时代追过
来源章节：004.哲学类.txt / 哲学类 / 严复：《侯官严氏丛刻》
主题标签：时代;自我更新;谦逊
李敖写严复曾是中国近代化变局中的先知，但后来时代走得比他快，他反倒落后。一个人曾经先进，不等于永远先进；若停止更新，过去的开明也会变成新的保守。人生要警惕把旧胜利当永久通行证。

【YBJZ028】救世要有身体代价
来源章节：004.哲学类.txt / 哲学类 / 墨翟：《墨子》
主题标签：承担;救世;行动
李敖写墨家摩顶放踵、赴火蹈刃，比高高在上的儒家救世更动人。说关心天下很容易，真正的考验是愿意付出多少身体、时间和风险。没有代价的救世，常只是姿态；肯下场，才有重量。

【YBJZ029】别把会评论误当会办事
来源章节：004.哲学类.txt / 哲学类 / 颜之推：《颜氏家训》
主题标签：办事能力;现实;读书
李敖引颜之推批评文学之士品藻古今却试用无所堪，不懂丧乱、战阵、耕稼和劳役。这一条重点不只是学问要关心现实，而是别把会评论误当会办事。文字上的高谈若不能转成对战乱、生产、劳役和民生的理解，遇到真实事务就会露怯；读书人要能把见识落到可处理的问题上。

【YBJZ030】与虎谋皮的知识分子，终会进虎口
来源章节：005.宗教类.txt / 宗教类 / 董仲舒：《春秋繁露》
主题标签：知识分子;权力;警惕
李敖写董仲舒用阴阳灾异逢迎主上，以期得君行道，结果成了专制政治和统一思想的帮凶，自己也险些被吞。知识分子若把真理交给权力照顾，常会先替权力做事，最后再被权力反咬。靠虎成事，最难保不入虎口。

【YBJZ031】反形式，是为了抓精神
来源章节：005.宗教类.txt / 宗教类 / 慧能：《六祖大师法宝坛经》
主题标签：形式;精神;目标
李敖写慧能反对繁琐佛经、口念般若、坐禅形式，重视精神上的开启。反形式不是不要纪律，而是不让形式冒充内容。人生中许多仪式、流程、规矩都可能有用，但若遮住本心和目标，就该被重新审问。

【YBJZ032】民间迷信也能读出民间呼声
来源章节：005.宗教类.txt / 宗教类 / 佚名：《太平经》
主题标签：民间;迷信;同理
李敖说《太平经》虽是道教早期经典，却能让人看到中国民间怎样追寻、怎样迷信、怎样提出呼声和要求。面对荒诞材料，不能只笑它荒诞；有时荒诞正保存了普通人的焦虑和愿望。读懂迷信背后的人，才算读深一层。

【YBJZ033】夸口长生，先过死亡这一关
来源章节：005.宗教类.txt / 宗教类 / 葛洪：《抱朴子》
主题标签：长生迷信;事实检验;理论
李敖写葛洪一生讲长生，最后八十一岁仍然死了，称这是中国长生理论家的最大幻灭。这一条不是普通的实践检验，而是提醒人别被承诺永生、万能、绝对有效的体系迷住：越是声称能解决终极问题的学说，越要拿最硬的事实来试。连自己的核心许诺都兑现不了，体系再漂亮也要打折。

【YBJZ034】好知己看处境，不只看结果
来源章节：006.社会科学类.txt / 社会科学类 / 管仲：《管子》
主题标签：识人;知己;处境
李敖引管仲回忆鲍叔牙，真正知己不因他多取财就说贪，不因谋事失败就说愚，不因临阵退走就说怯，而能看见贫穷、时势和老母。识人不能只按结果判刑；懂得把处境算进去，才配说理解一个人。

【YBJZ035】守旧常披着不事事的外衣
来源章节：006.社会科学类.txt / 社会科学类 / 王安石：《王临川集》
主题标签：改革;责任;行动
李敖写王安石回信司马光，不能接受“一切不事事，守前所为而已”。有些稳健其实只是怕动，有些传统只是怕负责。面对明显需要改革的局面，什么都不做也是选择，而且常是最保守、最有后果的选择。
