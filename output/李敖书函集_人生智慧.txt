条目数：58

【SHF001】搪塞要追到理由变形处
来源章节：001.跟警察头子打笔仗.txt / 跟警察头子打笔仗
主题标签：追问;逻辑;抗压
李敖把警察系统前后矛盾的说法一层层拆开：基层说上级有令，上级又说基层可斟酌，最后让借口自己露出破绽。对待敷衍，最有效的办法常常不是情绪化反击，而是持续追问同一个关键点，直到对方的理由在时间、层级和事实之间互相打架。

【SHF002】法律不能被魔术化解释
来源章节：002.查禁《孙中山研究》的荒谬理由.txt / 查禁《孙中山研究》的荒谬理由
主题标签：规则;法治;辨析
李敖要求当局说明，“孙中山卖国”这一标题究竟怎样落入出版法禁条，而不是任由权力把法条变成万能口袋。做人做事也一样，凡是牵涉规则，就要回到文字、事实和适用范围；不能让一个漂亮名词或严厉条文，随意变成压人的魔术。

【SHF003】制度换了名字，旧账仍要追问
来源章节：003.给新闻局局长邵玉铭的公开信.txt / 给新闻局局长邵玉铭的公开信
主题标签：责任;制度;追问
李敖指出，解严之后查禁机制从警总换到新闻局，并不等于旧问题自然消失。现实里许多压迫会换标牌、换机关、换话术，聪明的应对是抓住事实主线继续追问；只要损害还在，责任就不能因为制度改名而被冲淡。

【SHF004】把申诉变成留证的程序
来源章节：003.给新闻局局长邵玉铭的公开信.txt / 给新闻局局长邵玉铭的公开信
主题标签：证据;程序;坚持
李敖反复诉愿、申诉，不只是求一个即时结果，也是在迫使机关留下书面理由，让荒谬进入可检验的记录。人生里的申辩若只停在口头委屈，很容易被风吹散；把过程变成证据，把不公逼进档案，才有继续翻案和说服后来者的力量。

【SHF005】让内容先活下来
来源章节：004.许水德请注意.txt / 许水德请注意
主题标签：策略;传播;原则
李敖为了让《孙中山研究》继续流通，愿意暂时拿掉封面上被罗织为罪名的元素，但同时继续追究禁书的荒谬。坚持原则并不等于每一步都硬撞同一块石头；有时先拆掉对方最容易借题发挥的靶子，让内容活下来，才是更长久的抵抗。

【SHF006】受压时开出新战场
来源章节：005.致《中报》傅朝枢先生.txt / 致《中报》傅朝枢先生

李敖面对查禁、起诉和可能入狱，反而提出继续在海外写专栏、在岛内出书，让压制者付出更高解释成本。人被逼到窄处时，最怕只盯着那一扇被关上的门；真正有韧性的人会另开窗口，把压力转成扩大战场的动力。

【SHF007】公开伤害必须公开修补
来源章节：007.再致施性忠老哥.txt / 再致施性忠老哥
主题标签：道歉;名誉;责任
李敖拒绝“私下道歉”来解决公开诽谤，因为伤害是在公众面前发生的，修补也必须回到公众面前。人际关系里，诚意不能只挑对自己方便的场合出现；错误在哪里造成影响，就该在哪里承担恢复名誉和信任的责任。

【SHF008】承诺的善款也要兑现

主题标签：信用;承诺;自律
李敖在信中说明自己按承诺捐出书款三分之一，即使争执激烈，也不把公益承诺拿来讨价还价。一个人的信用常在小处显影：钱数可以不大，场面可以不热闹，但说过的话是否如数做到，决定了别人日后是否还敢把信任交给你。

【SHF009】知道自己的战场在哪里

主题标签：定位;自知;事业
李敖自称“书房里的战士”，宁可影响知识分子，也不为追逐群众场面而降低自己的表达方式。人生的成熟之一，是知道自己的力量适合在哪里发挥；不是所有人都要站上同一种舞台，找准战场，比到处迎合更能守住长期价值。

【SHF010】善事不必借神道包装

李敖拒绝用宗教色彩包装公益，认为不该以“神道设教”的方式拉人入局。做好事若要靠迷信、神秘或情绪勒索来扩大声势，反而会污染它的根；真正稳当的善意，应该经得起理性、透明和普通人的日常判断。

【SHF011】晚年更该补上直笔

李敖鼓励吴相湘立刻重写旧作，把过去不能直说或不敢直说的部分补出来。人生越到后来，越不该只维护旧名声；若有机会把经验、证据和判断重新写准，晚年的价值就不在圆滑收场，而在补上年轻时欠下的真实。

【SHF012】承担责任不要拖无辜下水

李敖要求检察官不要因查禁书籍牵连无辜的王自义，真正要办就冲着自己来。一个人若把伙伴、下属或弱者推到风口，来替自己的选择承受代价，便是失了担当；真正的硬气，是该自己扛的事不向无辜者转嫁。

【SHF013】公职良知不够就该离席

主题标签：公职;良知;选择
李敖对检察官说，若不能维持正义，就应辞职。任何位置都有它的最低道德门槛，尤其掌握别人命运的职位，更不能只把薪水、命令和升迁当作理由；当一个人明知不义却继续坐在位子上，他就已经参与了不义。

【SHF014】不合法不诚信的文件不能签
来源章节：011.从小事看认真.txt / 从小事看认真
主题标签：诚信;程序;自律
李敖拒绝在有问题的会议记录和支票上签字，哪怕只是大楼事务，也不把形式责任轻轻放过。签名不是一支笔划过去，而是把自己的信用押在那份文字上；凡是事实不清、程序不正、账目不明的文件，都不该用人情硬盖过去。

【SHF015】多数决不能被文字偷换
来源章节：011.从小事看认真.txt / 从小事看认真
主题标签：公共事务;程序;公正
李敖指出，十一人中五人同意不能写成通过，未表态也不能被冒充为无意见。公共事务最怕把数学问题写成文字游戏；尊重多数之前，先要尊重计数和表述，不能让会议记录把沉默、缺席和反对偷偷改造成同意。

【SHF016】改革要可执行，不靠高调
来源章节：011.从小事看认真.txt / 从小事看认真
主题标签：改革;执行;现实
李敖谈老旧大楼改革时，主张面对现实、快刀斩乱麻，而不是用漂亮话掩饰多年积弊。修补一个坏系统，最需要的是可执行的顺序、成本和责任，不是把问题说得庄严；口号越高，若没有方案，越容易成为拖延的烟幕。

【SHF017】好书赔钱也值得救
来源章节：012.复李声庭.txt / 复李声庭
主题标签：文化;价值;坚持
李敖愿意保留那些销量不佳却有自由、人权、民主价值的旧书，不让它们被浅薄畅销书淹没。人生里有些事不能只按眼前收益判断，真正有价值的东西常常先赔钱、先寂寞；愿意给它们留位置，就是在替未来保留养分。

【SHF018】亲历者要写无忌讳自传
来源章节：013.复李杰四弟.txt / 复李杰四弟

李敖劝长辈写回忆录，要细写个人浮沉、家族遭遇和时代变迁，不必回避难堪。一个时代若只留下胜利者的口号，后来的人就会失去真实坐标；亲历者把所见所受写出来，是把私人记忆变成公共见证。

【SHF019】苦难不能只说过去就算
来源章节：014.复六叔.txt / 复六叔
主题标签：苦难;记录;见证
李敖提醒六叔，遭遇血泪若不据实写出，就不是积极的收场。人生中的苦难当然可以不被仇恨绑架，但也不能被一句“都过去了”轻轻埋掉；把它写明、说透，才可能让伤痕变成经验，而不是白白受苦。

【SHF020】先真人真事，再文学加工
来源章节：014.复六叔.txt / 复六叔
主题标签：真实;写作;材料
李敖建议先写有真名、真事、真照片的自传，再考虑小说化处理。真实材料是一切改写的根，若根部空了，文采越好越可能变成漂浮；先把事实立住，后来的文学、评论和历史理解才有可靠的支点。

【SHF021】流传文化也要保留出处
来源章节：015.致余纪忠.txt / 致余纪忠
主题标签：出处;尊重;文化
李敖提醒报纸转载照片却不注明来源，会伤害提供者和相关人的感情。传播不是把材料拿来用完就算，越是珍贵的图文，越要保留来路、尊重托付；注明出处看似小礼节，实则是在维护文化流通中的信任链。

【SHF022】迟来的澄清也要逐项列证
来源章节：016.迟来的澄清.txt / 迟来的澄清
主题标签：澄清;证据;清白
李敖为一桩多年案件得到澄清后，仍逐条列出当初笔迹、记录、证物和程序上的问题。被误解的人最容易急着喊冤，但真正有力的澄清不是一句“我清白”，而是把错在哪里、错了多久、错由谁造成，一项一项摆出来。

【SHF023】一个人的事业要承认边界

李敖解释自己办报办刊时承受体力、时间和组织限制，不能把所有期待都独自吞下。理想越大，越要看清身体和组织的边界；硬撑可以赢得一时赞叹，却未必能让事业长久，懂得调整形式也是负责。

【SHF024】清音孤立也要力持原则

李敖借“浊世有清音”说明，真正坚持原则的人往往会越来越孤立，但仍要继续努力。并不是所有正确都会马上得到掌声，有时越清醒越寂寞；若明知环境浑浊还愿意守住清音，人的精神才没有被流俗完全收编。

【SHF025】在位就要谋其政

李敖谈社区职务时说，既然在其位，就必须履行职责，把积弊处理好。任何位置都不只是名分和方便，它同时带着该做的事；若只想享受位置带来的称呼，却不肯承担麻烦，所谓负责就成了空话。

【SHF026】公共费用不能靠好人无限补贴

主题标签：公共事务;公平;成本
李敖分析大楼费用分摊时，反对让少数识大体的人长期替别人承担成本。公共生活若总靠好说话的人垫付，就会奖励逃避责任的人；费用要算清、义务要分明，才不会让善意被制度性消耗。

【SHF027】别用官僚文字冷处理旧贡献者

李敖替母亲争取宿舍权益时，痛斥学校把退休老教师当皮球般处理。组织面对曾经贡献过的人，最能看出是否还有人味；一纸冷冰冰公文也许合乎流程，却可能暴露它已经忘了恩义、尊严和基本体恤。

【SHF028】珍惜时间远离空洞应酬

李敖说明自己很少参加婚丧喜庆等场合，不愿把时间耗在虚礼上。人要活出自己的分量，就必须敢于拒绝许多看似“应该去”的场面；把时间还给写作、思考和真正重要的人，才不会被世俗礼数磨成别人期待的样子。

【SHF029】荣辱标准因人格而异

李敖认为，把一个以反权势自许的人写成拥抱大官，本身就足以伤害名誉。评价一个人的荣辱，不能只看字面动作，还要看它如何冲撞此人的一贯人格；真正理解名誉，是理解一个人把什么视为底线。

【SHF030】标榜公正的人要接受测试

李敖把申诉递给新闻评议会，就是要看它是否能在有旧怨和强势媒体牵涉时仍守标准。公正不是写在章程上的自我介绍，而是在具体纠纷中被检验出来的；越是自称中立的机构，越不能害怕被拿个案测试。

【SHF031】更正要真实公开，不做特制品

【SHF032】接任位置也接任解释责任
来源章节：031.中央研究院史语所岂可掩护李济、许倬云！.txt / 中央研究院史语所岂可掩护李济、许倬云！

李敖追问中央研究院史语所旧案时，不接受后来者用“不知道来由”推托。接任一个职位，不只接收办公室和头衔，也接收该机构未了的账；不知道可以查，不能把无知当免责，因为位置本身就带着解释和补正的义务。

【SHF033】公帑成果不能藏成私物
来源章节：031.中央研究院史语所岂可掩护李济、许倬云！.txt / 中央研究院史语所岂可掩护李济、许倬云！

李敖质疑学术成果若使用公家资源，就不能长期以非卖品、赠阅品方式让公众难以取得。公共资源生产出的成果，应回到公共检验和公共使用；若变成少数圈内人的私相授受，知识就被制度性地锁进小圈子。

【SHF034】规则签收后不能暗中换
来源章节：032.书而不展，还叫什么书展？.txt / 书而不展，还叫什么书展？

李敖质问书展既已按规则收书，就不该临时撤书、暗中改标准。做事最基本的诚信，是事前规则和事后执行相一致；若名字叫书展却不让书展出，规则就从共同约定变成了临场操弄。

【SHF035】别为荒谬权力改写自己
来源章节：033.休做台北落荒人，应做云南望乡鬼.txt / 休做台北落荒人，应做云南望乡鬼
主题标签：尊严;权力;自我
李敖面对国民党式的荒谬限制，提醒人不要为迎合权力而自我阉割身份和尊严。外在规则若本身不合理，最糟的不是一时受阻，而是人在心里先替它让路；保住自己的判断，才不会被荒谬制度重新塑形。

【SHF036】旧学也要因新资料更新

李敖从旧诗句、旧掌故里又读出新的材料，显示熟悉并不等于已经穷尽。真正会读书的人，不把记住当作终点；即使是早年烂熟的东西，只要遇到新资料、新问题，也应该重新打开，让旧学继续生长。

【SHF037】权力旧习要持续盯防
