《李敖人生智慧》《波波颂》 人生智慧

条目数：70

【BBS001】作品靠整体水准，不靠入选光环
来源章节：《波波颂》自序.txt / 自序
主题标签：写作;自信;整体质量
真正有底气的创作，不必把价值押在少数被挑中的篇章上。若长期写作都有稳定水准，选本只是入口，不是镀金；做人做事也一样，最可靠的名片不是偶尔一两次漂亮表现，而是整体质量经得起抽查。

【BBS002】别把自己的感伤塞给自然
来源章节：001.对自然要自然.txt / 对自然要自然
主题标签：自然观;情绪;清醒
自然并不替人哭笑，风雨枯荣也不是专为个人情绪布置的舞台。看世界要学会把自己的小哀乐放回适当位置；少一点投射，多一点开阔，人才不会把短暂情绪误认为宇宙真相。

【BBS003】困境里也要保留感受力
来源章节：002.与阳光有约.txt / 与阳光有约
主题标签：困境;感受力;内在自由
人在封闭和匮乏中，最容易把心也关死。能从一束阳光里重新感到时间、空间和宇宙尺度，说明困境还没有夺走人的内在自由；感受力不是软弱，而是人在硬处境里保住尊严的细根。

【BBS004】时间变慢时学会大尺度生活
来源章节：003.时不我与，我不与时.txt / 时不我与，我不与时
主题标签：时间观;困境;忍耐
当分钟、钟点和日程都失去意义，人就不能再用平常的焦虑方式生活。被迫面对漫长时，要把自己从分秒得失里拔出来，用月和年衡量忍耐、修养和等待；大尺度有时是困境教出的本领。

【BBS005】不要用热闹表情审判真情
来源章节：004.用冰冷表现洁白.txt / 用冰冷表现洁白
主题标签：情感;克制;识人
有些真情并不表现为痛哭、拥抱和热烈言辞，反而可能显得冷静、克制，甚至不合俗套。判断一个人的情义，不能只看表情温度，更要看内里是否清白、可靠、经得起时间。

【BBS006】朋友宁缺毋滥
来源章节：005.朋友的没落.txt / 朋友的没落
主题标签：友谊;关系;自持
朋友本该使生命更高更宽，若只是消耗时间、制造牵累、把低水平的私情包装成义气，反而会成为负担。成熟的人不怕朋友少，怕的是把人生交给不值得的关系拖低。

【BBS007】友谊要靠人格和共同大事
来源章节：005.朋友的没落.txt / 朋友的没落
主题标签：友谊;人格;共同目标
旧相识、热闹往来和私人好处，都不足以支撑真正友谊。能长久同行的关系，必须有相近的人格底线，也最好有共同关心的大事；否则越亲密，越可能只是在彼此纵容。

【BBS008】把困境改造成观察现场
来源章节：006.活在湿度计里.txt / 活在湿度计里
主题标签：困境;观察;心境
环境若无法立刻改变，仍可以改变自己和环境相处的方式。把潮湿墙面看成一只奇异的仪器，把受困处境变成观察对象，人的心就不完全归困境支配。

【BBS009】剥夺会暴露依赖的真相
来源章节：008.论坐牢的好处.txt / 论坐牢的好处
主题标签：困境;依赖;自我认识
人平常以为离不开时间表、空间、朋友、对手和各种牵挂，真正被剥夺时，才看见哪些是生命所需，哪些只是生活习惯。极端处境的残酷之一，是它也会把依赖逐项拆开给你看。

【BBS010】伪君子比真小人多一层欺骗
来源章节：009.“显性伪君子”和“隐性伪君子”.txt / “显性伪君子”和“隐性伪君子”
主题标签：识人;伪善;道德
坏本身已经可厌，坏而假装成善更可厌，因为它把道德也拿来当掩护。识人不能只看对方口中的善意和姿态，还要看他是否用好话遮住坏事、用美名包装私利。

【BBS011】风凉话最容易伪装成正义
来源章节：009.“显性伪君子”和“隐性伪君子”.txt / “显性伪君子”和“隐性伪君子”
主题标签：伪善;批评;代价
站在安全处批评受苦的人，常显得理直气壮，实则只是把自己的无风险位置伪装成道德高度。真正的判断要先看说话者承担了什么代价；没有代价的高论，最该小心。

【BBS012】拉上窗帘不是仁慈
来源章节：009.“显性伪君子”和“隐性伪君子”.txt / “显性伪君子”和“隐性伪君子”
主题标签：同情;逃避;伪善
不看见别人的苦，并不会让苦消失，只会让自己继续舒服。很多伪善不是公开作恶，而是主动隔绝现实，好让良心不必被打扰；一个人越会装作没看见，越要问他在回避什么。

【BBS013】美德造型不能代替美德本身
来源章节：009.“显性伪君子”和“隐性伪君子”.txt / “显性伪君子”和“隐性伪君子”
主题标签：美德;形象;伪善
节俭、慈悲、清高、公益，都可能被做成供人观看的造型。真正的美德要落实到选择和代价上，不能只看衣着、口号和公共姿态；道德若只是造型，越漂亮越危险。

【BBS014】远方慈善不能遮住近处责任
来源章节：010.“三毛式伪善”和“金庸式伪善”.txt / “三毛式伪善”和“金庸式伪善”
主题标签：慈善;责任;伪善
关心远方受苦的人当然可贵，但若借远方悲悯逃开身边的不义，就成了漂亮的转移。善意要经得起远近两端的检验；能看见异乡苦难，也该看见门口的人。

【BBS015】选择性信仰只是自利
来源章节：010.“三毛式伪善”和“金庸式伪善”.txt / “三毛式伪善”和“金庸式伪善”
主题标签：信念;原则;自利
只挑对自己有利的教义、原则和价值来相信，表面像有信念，实则只是把信念当工具。真正的信仰和原则，恰恰要在不方便、不讨好、不赚钱时仍然约束自己。

【BBS016】有些危险不能因为危险就避开
来源章节：013.人到底该怎么选择？.txt / 人到底该怎么选择？
主题标签：选择;勇气;价值
趋利避害是常情，但人生并不只由安全计算构成。若是非、尊严和信念都在危险处，退一步也许保住身体，却丢掉了更重要的自己；选择要问风险，也要问价值。

【BBS017】别只劝受压者收声
来源章节：013.人到底该怎么选择？.txt / 人到底该怎么选择？
主题标签：正义;责任;直言
看见有人因直言受难，旁观者常劝他说话小心，却少劝施压者停止作恶。这样的劝告看似现实，实则把责任悄悄推给受害的一方；真正讲理，要把压力源也摆上桌。

【BBS018】别把改变寄托在拖延者的支票上
来源章节：014.爱丽丝与新党.txt / 爱丽丝与新党
主题标签：改变;拖延;判断
既得利益者最擅长开出遥远承诺，让人把今日行动押到未来兑现。判断改革诺言，不只听它说什么，更要看说话者是否从拖延中获利；若拖延本身就是利益，支票多半不会兑现。

【BBS019】崇拜以前先读全材料
来源章节：015.索尔仁尼琴错在哪儿？.txt / 索尔仁尼琴错在哪儿？
主题标签：阅读;判断;去神化
对名人的失望，常来自自己原先了解得太少。若一个人的作品、言论和立场早有痕迹，就不该等到偶像破裂才惊呼意外；少一点神化，多一点阅读，判断会稳得多。

【BBS020】反对一种恶，不等于拥抱另一种恶
来源章节：015.索尔仁尼琴错在哪儿？.txt / 索尔仁尼琴错在哪儿？
主题标签：原则;阵营;判断
一个人反对某种压迫，并不自动证明他站到所有正义一边。人生判断最怕用敌人的恶替朋友的恶开脱；真正的原则，要能同时拒绝两边的坏，而不是在阵营之间选盲点。

【BBS021】看真实不能只看安排好的行程
来源章节：015.索尔仁尼琴错在哪儿？.txt / 索尔仁尼琴错在哪儿？
主题标签：现实;观察;真相
被带去看花、看会场、看体面人物，不等于看见真实。想理解一个地方，要看被遮住的监狱、泥地、穷人和沉默者；凡是只让你看漂亮路线的地方，最该问路线外有什么。

【BBS022】象征人物更要爱惜原则
来源章节：015.索尔仁尼琴错在哪儿？.txt / 索尔仁尼琴错在哪儿？
主题标签：名望;原则;责任
当一个人已经成了公共象征，他的站位就不再只是私人意见。名望越大，越要谨慎自己替谁背书、和谁同框、被谁利用；原则受损时，伤的不只是个人声誉，也是许多人托付的信任。

【BBS023】信仰也要有辨别力
来源章节：016.别弄拧了堂吉诃德.txt / 别弄拧了堂吉诃德
主题标签：信仰;辨别力;现实
纯粹的信仰值得尊重，但若分不清风车和巨人、朋友和敌人，勇敢就可能变成荒唐。真正成熟的信念，不是闭眼冲锋，而是在热情里保留辨认现实的能力。

【BBS024】知其不可与不知其不可不同
来源章节：016.别弄拧了堂吉诃德.txt / 别弄拧了堂吉诃德
主题标签：坚持;清醒;判断
明知艰难仍选择去做，是承担；根本没看清现实却鲁莽去做，是糊涂。不要把所有失败的坚持都浪漫化，先分清它背后是清醒的责任，还是误判后的固执。

【BBS025】人不必永远做战士
来源章节：017.我是圣人.txt / 我是圣人
主题标签：人生阶段;战斗;自我照顾
一个人曾经奋斗、受难、承担，并不表示余生都必须保持同一种战斗姿态。人生有代谢，也有退场和转换；允许自己在完成一段使命后重新生活，并不等于背叛过去。

【BBS026】有人冲出网罗，别用痛苦绑住他
来源章节：017.我是圣人.txt / 我是圣人
主题标签：自由;友爱;困境
留在困局里的人，容易把逃出去者的自由看成亏欠。可真正的友爱应当为对方脱身高兴，而不是要求他继续用痛苦证明忠诚；别人能够重新生活，也是一种共同胜利。

【BBS027】道歉要屈于理，不该屈于力
来源章节：018.论“向将军道歉”.txt / 论“向将军道歉”
主题标签：道歉;原则;权力
真正的道歉来自明白自己错在哪里，而不是被权势压到低头。若一个人只是因为对方有官威、拳头或资源才道歉，那不是修复关系，而是把是非让位给力量。

【BBS028】错了要负责，但别交成降书
来源章节：018.论“向将军道歉”.txt / 论“向将军道歉”
主题标签：责任;尊严;边界
承认错误不等于无限让渡尊严。该赔的赔、该改的改，但不能把责任扩大成屈服，把道歉变成投降；成熟的负责，是在事实范围内负责，不让权力趁机追加羞辱。

【BBS029】公权力别拿私人面子当公务
来源章节：018.论“向将军道歉”.txt / 论“向将军道歉”
主题标签：公权力;边界;制度
私人争端一旦被权力机器接管，就很容易从讲理变成示威。一个位置越高的人，越要分清个人不快和公共职权；用公器护面子，最后伤到的是制度的可信度。

【BBS030】用笔如剑也要承担代价
来源章节：019.皮尔逊与黑资料.txt / 皮尔逊与黑资料
主题标签：写作;勇气;代价
把文字当武器，就要预料诉讼、报复、监视和围攻。若只想享受揭弊的荣耀，不愿承担后果，笔就不会真正锋利；公共写作的勇气，包含对代价的清醒计算。

【BBS031】别替偷渡进来的目标孵蛋
来源章节：020.别孵那贼蛋！.txt / 别孵那贼蛋！
主题标签：边界;警觉;组织
一个组织、平台或人生计划，最怕被别人悄悄塞进不属于你的目标。发现有人借你的资源孵化他的结果，就要及时识别、拒绝和清理；善良若没有警觉，容易替敌意出力。

【BBS032】结构被污染时宁可重筑一层
来源章节：020.别孵那贼蛋！.txt / 别孵那贼蛋！
主题标签：风险;重建;边界
当旧结构已经被掺入危险因素，继续原样运作可能只会养大后患。必要时宁可增加成本、重建隔离层，也不要为了省事把隐患当成正常成员留下。

【BBS033】不骗自己以后还要重建希望
来源章节：022.江南，根本没有春天.txt / 江南，根本没有春天
主题标签：希望;清醒;现实
看破幻梦只是第一步，若只停在失望，就仍被梦支配。真正清醒的人不但不骗自己，还能在满脸风霜的现实里重新参与、修补和期待；希望要从醒后开始重建。

【BBS034】抒情高分不等于思想高分
来源章节：022.江南，根本没有春天.txt / 江南，根本没有春天
主题标签：阅读;思想;判断
文字可以非常动人，判断却仍可能软弱或逃避。读文章不能只被情绪带走，还要问它有没有面对现实、承担责任、给出清醒方向；好文采不自动等于好思想。

【BBS035】爱也要承认生活秩序成本
来源章节：023.波波颂.txt / 波波颂
主题标签：爱;责任;生活秩序
喜欢一件事物，不等于一定适合拥有它。真正诚实的爱，会把照料、时间、工作秩序和生活代价一起算进去；只说喜欢而不管后果，常常只是把爱讲得太轻。

【BBS036】真喜欢要落到持续照料
来源章节：023.波波颂.txt / 波波颂
主题标签：爱;照料;日常
口头说爱很容易，真正困难的是长期喂养、清理、让位和适应。喜欢若不能变成稳定照料，就还只是情绪；能被日常麻烦考验过的爱，才比较可信。

【BBS037】本色胜过装扮
来源章节：024.能下床就是好猫.txt / 能下床就是好猫
主题标签：本色;行动;识人
外表、称号和角色都可能改变，但一个人真正会在关键动作里露出本色。与其迷信包装，不如看他本能地追什么、守什么、忘不忘本职；行动比装扮更接近真相。

【BBS038】好古还要敏求
来源章节：026.古董与今董.txt / 古董与今董
主题标签：求知;传统;辨伪
喜欢传统、古物或旧学，不该只停在摆设和姿态上。真正的爱好要肯查、肯问、肯分辨，不然越热衷越容易被骗；热爱若没有求知，只是换了古雅名字的虚荣。

【BBS039】明知是假，反而玩得清醒
来源章节：026.古董与今董.txt / 古董与今董
主题标签：清醒;趣味;辨伪
仿品、游戏和替代物未必不能欣赏，关键是别骗自己也别骗别人。知道它是假的，还能从工艺、趣味和当下快乐中取乐，这叫清醒；把假的当真的炫耀，才叫迷失。

【BBS040】人间天堂不能硬造
来源章节：027.天堂不是我们的.txt / 天堂不是我们的
主题标签：理想;现实;方法
越是声称要马上造出天堂的计划，越要警惕它用什么方法。好目标若配上强迫、幻想和对现实复杂性的轻视，往往会从开头就走错；真正改善人间，比宣布天堂艰难得多。

【BBS041】责任未尽，天堂也不是归宿
来源章节：027.天堂不是我们的.txt / 天堂不是我们的
主题标签：责任;理想;行动
个人安逸再诱人，若眼前仍有该面对的黑暗和责任，就不能急着把自己安放进乐园。成熟的理想不是逃向洁净处，而是在不洁净的人间继续做该做的事。

【BBS042】要站到被间接伤害者那边看一眼
来源章节：029.美国人忘了什么？.txt / 美国人忘了什么？
主题标签：历史;责任;同理
一个人未必亲手作恶，却可能支持、纵容或受益于别人手里的刀。判断历史和现实，不能只站在自己的善意叙事里，也要站到被自己间接伤害的人那边，看他们记住了什么。

【BBS043】自由不能只供自己庆祝
来源章节：030.神像百年一感.txt / 神像百年一感
主题标签：自由;原则;一致性
若只在自己家里赞美自由，却在别人那里纵容压迫，自由就变成了装饰而不是原则。真正的价值要能跨过边界、盟友和利益考验；只对自己有效的自由，可信度有限。

【BBS044】用坏政权求苟安，最后苟安也不可得
