【TZHA001】先知不是嘴硬，是提前留证
来源章节：001.19990928挑战李敖.txt / 19990928挑战李敖
主题标签：预见;证据;判断
李敖回看自己十年前写书判断人物，不把“我早就说过”当成口号，而是拿书、判决书和资料来证明。人生要有预见，不是靠嗓门比别人早，而是把判断建立在可回头检验的证据上；时间过去以后，材料仍站得住，才叫看得远。

【TZHA002】能救问题的力量，不必先问阵营
来源章节：001.19990928挑战李敖.txt / 19990928挑战李敖
主题标签：务实;救助;成见
谈灾后复建时，李敖主张只要经验和力量能帮上忙，就不该因为来自哪一边而拒绝。人生处理大问题，先问方法是否有效、资源是否可靠、结果是否有利；若把阵营好恶摆在前面，常会错过真正能解困的手。

【TZHA003】用大尺度校准标准
来源章节：001.19990928挑战李敖.txt / 19990928挑战李敖
主题标签：标准;参照系;反省
李敖批评台湾的政治评价落到小地方标准，忘了世界级和古今尺度。人生越在局部环境里待久，越要主动拉高参照系；只跟身边人比，容易把狭隘当成熟，把低标准当正常。

【TZHA004】反对坏话，也别反掉言论自由
来源章节：002.19990929挑战李敖.txt / 19990929挑战李敖
主题标签：言论自由;选择;边界
李敖反对政府责任被掩盖，也反对号召别人不看某媒体这种干涉选择的做法。人生维护价值时，最难的是不让自己的正义感变成新的压迫；批评别人可以尖锐，但不能把别人的听闻、选择和表达一并没收。

【TZHA005】别把外在禁令变成心里的禁令
来源章节：003.19991001挑战李敖.txt / 19991001挑战李敖
主题标签：自我审查;勇气;自由
李敖用“悬崖理论”说明，人怕碰到权力红线，常会在心里先设一条更退缩的线。人生真正可怕的限制，常不是别人明文禁止你，而是你为了安全不断自我阉割；久了以后，刀还没落下，人已经先矮了一截。

【TZHA006】好人难做，尤其要做给被遗忘的人
来源章节：004.19991004挑战李敖.txt / 19991004挑战李敖
主题标签：善意;第一线;被遗忘者
李敖谈救灾时注意到，跑到边远地区第一线的人反而容易被误解。人生做好事不要只挑聚光灯下的对象，也不要因为可能挨骂就退回安全处；真正的善意，常要去看那些人数少、地方远、没人替他们说话的人。

【TZHA007】法律没写到，不等于良知允许
来源章节：005.19991005挑战李敖.txt / 19991005挑战李敖
主题标签：规则;分寸;底线
李敖借拳赛打点滴和旁听席布条的例子，说明不能把规则没有明说当作什么都可以做。人生许多底线靠常识、惯例和互相尊重维持；只会钻条文空白的人，也许赢了文字，却输掉了做人分寸。

【TZHA008】成熟社会靠不在话下的约束
来源章节：005.19991005挑战李敖.txt / 19991005挑战李敖
主题标签：惯例;成熟;自治
李敖谈英国国会时强调，真正自由民主的社会并不是什么都靠明文规定，而靠大家心照不宣地遵守惯例。人生的成熟也一样：越需要把每件事写死、吵透、逼到规则边缘，越说明彼此的信任和修养还不够。

【TZHA009】人格不要按族群分类
来源章节：006.19991006挑战李敖.txt / 19991006挑战李敖
主题标签：人格;标签;识人
李敖把许多身份标签拆开，指出真正可分的不是本省外省，而是聪明与愚蠢、要脸与无耻。人生识人若先看出身，常会被标签带着走；真正可靠的判断，要落到行为、胆识、责任和人格。

【TZHA010】找对人做事，比亲自做秀有用
来源章节：007.19991007挑战李敖.txt / 19991007挑战李敖
主题标签：授权;做事;识人
李敖说若自己有权力，会找最该做事的人去做，而不是凡事亲自管、亲自露面。人生负责不是把自己摆进每个镜头，而是把资源交给真正懂行的人；会授权、会识人，往往比忙着表现更有力量。

【TZHA011】走出悲情靠行动，不靠仪式

主题标签：悲情;行动;创痛
李敖反感大场面追悼一边制造悲情，一边口头说走出悲情。人生面对创痛，仪式可以安顿情绪，却不能替代修补、负责和改变；若只不断放大哀伤，反而会把人锁在悲情里。

【TZHA012】要好制度，不要好人保护
来源章节：010.19991012挑战李敖.txt / 19991012挑战李敖
主题标签：制度;保障;好人
李敖明确说，不要好人来保护我们，要好的制度来保护我们。人生不能把安全感押在某个好人、明君或贵人身上；人会变、会老、会失误，只有制度、流程和边界能把善意变成可持续的保障。

【TZHA013】专业角色别背过度道德要求

【TZHA014】别把自己装成完人去监管别人

李敖批评一些人以完美人格姿态出来管钱管事，好像别人都可疑。人生要参与公共事务，最好带着制度意识而不是圣人姿态；承认自己也可能有私心和盲点，才不会把监督变成自我神化。

主题标签：时间;时机;代价
石滋宜提醒，时机就是一切，时间过去后再争回来，代价不知要多少。人生最贵的成本常不是钱，而是错过窗口；看见趋势却迟迟不动，后来再补，往往要用数倍力气。

【TZHA086】长期掌权的人要换位反省
来源章节：107.20000303挑战李敖.txt / 20000303挑战李敖
主题标签：反省;权力;换位
节目中谈到真正民主的发展，是让执政者到在野位置设身处地反省。人生无论在团队还是组织里，长期握有资源都会失去体感；偶尔站到没有资源的位置，才知道规则是否公平、基层是否辛苦。

【TZHA087】感情不要一次用光
来源章节：108.20000304挑战李敖.txt / 20000304挑战李敖
主题标签：感情;节制;余味
李敖对学生说，感情要有所扩散、有所约束，保持一点余味，不要用光。人生的热情和爱都需要节制；一开始燃尽，看似真诚，后来只剩疲惫，能留余味的人，关系反而更长久。

【TZHA088】追真相不能停在替罪羊面前
来源章节：109.20000313挑战李敖.txt / 20000313挑战李敖
主题标签：真相;替罪羊;追责
李敖谈军购回扣案时，注意到有人被推成挡箭牌，真正的大人物反而可能躲在后面。人生查问题不要满足于抓到一个最容易被牺牲的人；替罪羊出现时，更要问谁从遮挡中得利。

【TZHA089】还在场，就还有机会
来源章节：110.20000330挑战李敖.txt / 20000330挑战李敖
主题标签：坚持;机会;翻盘
李敖借许信良坚持参选，说明只要还在局中，就仍可能赢。人生有些局面不到最后不必急着退场；坚持不是盲目硬撑，而是在还握有位置和时间时，保留翻盘的可能。
