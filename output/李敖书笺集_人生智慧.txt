李敖书笺集 人生智慧

条目数：82

【SJIAN001】用不完美的人制衡更坏的局面
来源章节：001.我为什么支持王八蛋？.txt / 我为什么支持王八蛋？
主题标签：判断;方向;现实
李敖说支持反对党政治，不是因为反对派个人多高尚，而是为了反对一党独大、一党独裁。现实里有时不是在圣人与小人之间选择，而是在坏格局中寻找制衡力量；看得见对方不完美，仍能为了更大平衡而有限支持，才是不幼稚的判断。

【SJIAN002】被列黑名单时，要追问系统来源
来源章节：002.斥财阀.txt / 斥财阀
主题标签：黑箱;追问;机构
李敖面对信用卡公司忽称他在黑名单中，没有只争一笔消费，而是追问这种资料从哪里来，是否与警备总部或调查局相连。遇到机构性歧视，别只接受柜台说法；要追到名单、资料流和责任链，否则一次误会背后可能藏着一整套黑箱。

【SJIAN003】道歉若避开问题，就还没道歉
来源章节：002.斥财阀.txt / 斥财阀
主题标签：追问;道歉;问题
国泰信托只说职员误会并补发信用卡，却没有回答李敖追问的核心问题：黑名单从何而来、收费凭什么。人生中遇到轻飘飘的歉意，不能只看态度软不软，而要看它有没有处理真正的问题；绕开要害的道歉，只是拖延。

【SJIAN004】对死者的承诺，也要向生者兑现
来源章节：003.谁使雷震有了遗恨？.txt / 谁使雷震有了遗恨？
主题标签：承诺;信用;公道
李敖写信给宋英，要求照雷震遗意处理房子，不让死者地下有憾、生者权益受损。承诺不因当事人去世就失效，尤其涉及遗愿、财物和信任时，更要把话做成事；一个社会的人心公道，常从这些身后小处维系。

【SJIAN005】压死主流，往往只会养出乱流
来源章节：004.从主流到乱流.txt / 从主流到乱流
主题标签：异议;格局;后果
李敖回忆《文星》被掐死时曾说，尚有规格的主流反对若被消灭，日后只会变成更难收拾的乱流。处理不同意见也是如此：把有规则、有水准的异议压掉，并不等于问题消失，只是把它推向更粗暴、更无序的出口。

【SJIAN006】旧名称不能只靠旧情保管
来源章节：004.从主流到乱流.txt / 从主流到乱流
主题标签：权利;名称;检验
李敖请第三者登记《文星》，意在证明萧孟能不能只靠旧关系就宣称独占这个名称。记忆中的招牌不等于法律上的权利，情感上的贡献也不等于永久控制权；越是有历史光环的名字，越要经得起事实和权利的检验。

【SJIAN007】被抹黑时，要把平反追成记录
来源章节：004.从主流到乱流.txt / 从主流到乱流
主题标签：名誉;记录;反击
萧孟能指控李敖伪造文书，媒体大登特登；法院裁定李敖清白后，报纸却一字不登。李敖没有只靠愤怒，而是把裁定、判决和诽谤责任一一追出来。名誉受损时，最有力的反击不是口头喊冤，而是把真相追进正式记录。

【SJIAN008】特立独行，才不被两边劫持
来源章节：005.“望君早归”不如“望君牺牲”.txt / “望君早归”不如“望君牺牲”
主题标签：独立;领导;原则
李敖称赞江鹏坚退席抗议有新气象，并说只有特立独行，才不会被国民党劫持，也不会被党外浑人劫持。人在团体中最容易被“自己人”和“敌人”同时拖走；能守住独立姿态，才可能领导群众，而不是被群众推着走。

【SJIAN009】怕监牢的人，会被监牢支配
来源章节：005.“望君早归”不如“望君牺牲”.txt / “望君早归”不如“望君牺牲”
主题标签：勇气;代价;自由
李敖引用甘地，强调抗暴者若怕入狱，政府就能利用这种恐惧；把入狱看成受苦和净化，力量反而更大。人生中的许多威胁也是如此：你越把它当作不能碰的极限，它越能支配你；先在心里接受代价，行动才会自由。

【SJIAN010】别把大是大非唱成小儿女调
来源章节：005.“望君早归”不如“望君牺牲”.txt / “望君早归”不如“望君牺牲”
主题标签：牺牲;是非;成熟
李敖批评党外只会“望君早归”，不愿面对真正牺牲，结果只剩粉拳绣腿和小是小非。面对严肃问题时，温情不是不能有，但不能替代大判断；若只求安慰和团圆，反而会把本该承担的正义责任唱软、唱小。

【SJIAN011】别人的胜利，也要承认为胜利
来源章节：006.手下·下手·快.txt / 手下·下手·快
主题标签：公正;评价;派系
李敖不满党外舆论把施性忠当选说成不算党外胜利，却把康宁祥落选说成党外失败。评判事情要有一致标准，不能让派系好恶决定输赢定义；不肯承认别人的成绩，只会暴露自己的小气和不公。

【SJIAN012】迎合者常把命令扩大成伤害
来源章节：006.手下·下手·快.txt / 手下·下手·快
主题标签：迎合;权力;警惕
查禁命令还没到，地方人员已先“借阅”、打招呼、扣书，李敖讽刺这是“手下·下手·快”。很多伤害不是最高命令直接造成，而是迎合者抢先加码造成；与组织打交道，既要看上面的规矩，也要防下面的邀功。

【SJIAN013】救济不能替责任者擦屁股
来源章节：007.党外不可帮国民党擦屁股.txt / 党外不可帮国民党擦屁股
主题标签：救济;责任;追责
李敖反对党外把义卖所得直接捐给矿灾受害者了事，因为灾变责任在国民党政府，不能让善心替责任者卸责。帮助别人不只是给钱，更要看钱有没有遮住真正的责任；若救济让作恶者轻松脱身，善意也会变成配合。

【SJIAN014】送火把有时比送炭重要
来源章节：007.党外不可帮国民党擦屁股.txt / 党外不可帮国民党擦屁股
主题标签：权利;行动;扶助
李敖主张义卖所得应用来组织赔偿委员会、请律师、办宣传、追究国家赔偿，而不只是分发爱心。真正扶助弱者，不是让他们暂时暖一下，而是给他们争权利的工具；有些时候，点燃行动的火把比一袋炭更要紧。

【SJIAN015】假货一露馅，致敬就该取消
来源章节：008.“致敬取消了！”.txt / “致敬取消了！”
主题标签：识人;事实;修正
李敖写军法处里年轻人误把李荆荪当共产党前辈，后来发现是假的，便传话“致敬取消了”。尊敬不能靠传说和标签维持，一旦事实证明对象不配，就要有撤回敬意的勇气；承认看错，比继续捧假货更体面。

【SJIAN016】真左真右，都要看本事和诚意
来源章节：008.“致敬取消了！”.txt / “致敬取消了！”
主题标签：标签;识别;诚意
李敖讽刺台湾一些号称左派的人，其实连共产党一根毛都没摸到，只剩生硬文章和圈内瓜分。任何标签若没有知识、纪律和诚意支撑，都只是姿态；判断人不要听他自称站在哪边，要看他是否真懂、真做、真承担。

【SJIAN017】有一分证据，才说一分话
来源章节：009.从告自己人的密到缴自己人的枪.txt / 从告自己人的密到缴自己人的枪
主题标签：证据;判断;史识
胡虚一交出雷震电文底稿，李敖据此讨论史事，重点正是胡适所谓“有一分证据，说一分话”。判断过去或评价他人，最忌拿印象补材料；手上有多少证据，就说到哪里，证据之外宁可留白。

【SJIAN018】告自己人的密，最能看出组织病
来源章节：009.从告自己人的密到缴自己人的枪.txt / 从告自己人的密到缴自己人的枪

李敖从曹圣芬向蒋介石打小报告谈起，指出国民党“内斗内行”、倾轧取媚、陷害同志。一个组织若总靠密告和邀宠来分配安全感，它就会先消耗自己人；真正危险的不是外敌多强，而是内部把互害当本事。

【SJIAN019】危机中先缴自己人的枪，是信任破产

主题标签：信任;危机;组织
刘汝明部队撤台后，被扣在船上不给饭水，最后徒手上岸、拆散收编。李敖看出这比战败更能说明国民党性格：危机中若先解除自己人的能力，而不是照顾他们，就等于宣布共同体已经没有信任。

【SJIAN020】看清换边也未必有出路
来源章节：009.从告自己人的密到缴自己人的枪.txt / 从告自己人的密到缴自己人的枪
主题标签：投机;处境;自持
李敖说刘汝明回忆录里的书面口气，与他听到的口语大不相同：跟国民党这么多年都随时会被丢掉，换到共产党那里也未必有人要。所谓不投机，有时不是漂亮姿态，而是看清投机也未必有出路；在烂局里，少做一次投机，至少还能守住自己。

【SJIAN021】没人敢写的暗账，笔杆要写
来源章节：009.从告自己人的密到缴自己人的枪.txt / 从告自己人的密到缴自己人的枪
主题标签：写作;历史;责任
李敖说枪杆在他们手里，笔杆在我们手下；发掘国民党无情史与下流史，是义不容辞的事。面对被遮蔽的历史，写作不是装饰，而是一种清账；别人不愿写、不敢写，正是你该写的理由。

【SJIAN022】先问规则管不管你这类人
来源章节：010.捞过了界的扣押.txt / 捞过了界的扣押
主题标签：规则;适用;维权
台北市政府以“未经出版登记”为由扣书，李敖没有先默认自己违规，而是查出登记要求针对“出版业”，著作人直接出书另有排除规定。遇到规则压人，第一步是问这条规则本来管谁；适用对象错了，处分从根上就站不住。

【SJIAN023】人的规格会被土地和教化塑形
来源章节：011.政治规格的讨论.txt / 政治规格的讨论
主题标签：环境;人格;判断
李敖回应邓维桢时说，政治规格既有中国人的“大同”，也有台湾地区“土地教化”造成的“小异”。看人不要只看个性，也要看他长期被什么环境训练；许多软弱、献媚和混乱，不是突然出现，而是土壤慢慢养成。

【SJIAN024】受害者若配合权力，权力就名利全收
来源章节：011.政治规格的讨论.txt / 政治规格的讨论
主题标签：受害;解释权;警醒
李敖批评林义雄出狱后的恭顺姿态，认为国民党既抓人、关人、杀人，又得到宽大的形象和受害者的感谢，等于名利全收。受过伤害的人若在关键处替伤害者圆场，常会让原本的不义被包装成恩德；越有苦难记录，越要守住解释权。

【SJIAN025】唤醒梦中梦，要耐得住孤寂
来源章节：011.政治规格的讨论.txt / 政治规格的讨论
主题标签：孤独;提醒;使命
李敖在日月潭看到“唤醒梦中梦”的联语，自比夜半钟声，对做梦的人苦口婆心。他知道这是“千山我独行”的孤寂事业。真正想提醒时代的人，不能指望立刻被感谢；钟声先要响，醒不醒常在以后。

【SJIAN026】反击失实，要排出时间线
来源章节：012.胡茵梦的“天性”问题.txt / 胡茵梦的“天性”问题
主题标签：澄清;证据;时间线
面对报纸报道和胡茵梦证词，李敖没有只说对方说谎，而是按存证信、法庭笔录、后续证词列出前后矛盾。澄清复杂争议时，时间线是最好的武器；让事实按顺序站出来，胜过一百句情绪辩解。

【SJIAN027】清高若前后不一，就只是姿态
来源章节：012.胡茵梦的“天性”问题.txt / 胡茵梦的“天性”问题
主题标签：一致;清高;人格
李敖指出胡茵梦三年前说不接受房子赠与，三年后又说有权处分财产，前后完全矛盾。一个人是否清高，不看他某一刻怎么表态，而看利害变化后是否仍守同一说法；前后不一的清高，只是方便时的姿态。

【SJIAN028】雪中送炭，只看兑现不听保证

主题标签：情义;行动;承诺
李敖回信说，别人说要在危难时公开支持，不可轻信，要真正送了才算。危机前的豪语最便宜，危机中的行动最昂贵；判断情义，不要记人说过多少漂亮话，要看风雪来时他有没有出现。

【SJIAN029】能戒掉依赖，才见意志
来源章节：013.雪中送炭与雪中送肉.txt / 雪中送炭与雪中送肉
主题标签：自制;意志;习惯
李敖说自己戒烟戒酒近二十年，因此看不起不能戒烟戒酒的人，认为靠几口酒、一片烟雾维持稳定，是意志和道性不足。自制不是苦行姿态，而是证明自己不被小依赖牵着走；戒得掉，人才有更大的自主。

【SJIAN030】程序睡着了，就把它叫醒
来源章节：014.是李敖就没完的！.txt / 是李敖就没完的！
主题标签：程序;坚持;追问
李敖追问胡茵梦背信侵占案为何在法院和检方之间沉睡三年，引用令函要求说明。制度里很多不公不是公开拒绝，而是拖、漏、睡；面对这种沉默，要把程序节点一一叫醒，不能让遗漏成为结案。

【SJIAN031】拆诡辩，要逐点分清问题
来源章节：015.驳国民党议员的四点胡说.txt / 驳国民党议员的四点胡说
主题标签：辩论;清晰;法律
郁慕明提出四点辩解，李敖逐条分清篇幅多少、更正函、刊登次数、发行人责任各自对应什么法律问题。遇到混在一起的辩解，不要被气势带走；把每一条拆回它真正相关的地方，诡辩就会失去烟雾。

【SJIAN032】责任身份不能随忙碌暂停
来源章节：015.驳国民党议员的四点胡说.txt / 驳国民党议员的四点胡说
主题标签：责任;身份;担当
李敖指出发行人就是发行人，法律上没有因助选、指定代理人或未涉编务就自动中止责任的规定。人生中许多责任也一样：你占着某个位置，就不能在出事时说自己那阵子很忙；名义带来权力，也带来责任。

【SJIAN033】先看对方举的到底是什么证
来源章节：019.驳国民党议员老婆的一片胡说.txt / 驳国民党议员老婆的一片胡说
主题标签：证据;辨析;责任
《秋海棠》说已举出事证证明李敖唯利是图，李敖翻出庭讯内容，指出所谓事证其实只是郁慕明把责任推给老婆。别人宣称“有证据”时，不要先被名词吓住；要看证据证明的是事实本身，还是只证明了对方如何逃责。

【SJIAN034】维护人权，不等于维护遁词
来源章节：020.维护人权与维护正义.txt / 维护人权与维护正义
主题标签：人权;真相;正义
李敖提醒周灿雄，替吴敦辩护当然要维护其人权，但对他的遁词，应按林肯、甘地等律师原则，不予支持。替一个人争程序保障，不代表替他的谎话背书；真正的正义，是同时保护权利和真相。

【SJIAN035】看见被利用者，也别忘真正受害者
来源章节：020.维护人权与维护正义.txt / 维护人权与维护正义
主题标签：同情;责任;案件
李敖说吴敦等被唆使于先、出卖于后，自有可怜处；但更可怜的是被杀的刘宜良。面对复杂案件，不能只看某一方的悲情而忘记伤害链条的终点；同情可以分层，责任不能因此模糊。

【SJIAN036】背后诉苦不能制止公开横行
来源章节：021.十日新谈.txt / 十日新谈
主题标签：勇气;揭发;分寸

【SJIAN075】社会责任可以和人际疏离并存
来源章节：040.爱因斯坦的两段话.txt / 爱因斯坦的两段话
主题标签：责任;距离;个性
李敖翻译爱因斯坦的话，深有同感：对社会正义和责任有热情，却对直接结社、人群、家庭保持疏离。一个人不热衷社交，不等于没有公共心；有些人的责任感恰恰需要距离来保存。

【SJIAN076】内心平静别押在别人意见上
来源章节：040.爱因斯坦的两段话.txt / 爱因斯坦的两段话
主题标签：内心;独立;平静
爱因斯坦说孤立虽苦，却让自己独立于习俗、物议和偏见，不把内心平静安放在浮动基础上；李敖特地译给朋友。别人理解与同情当然可贵，但若心安全靠外界供应，就会随风摇摆；真正的平静要有自己的地基。

【SJIAN077】做可靠朋友，也做难缠对手
来源章节：041.彼以熊来，吾以熊往.txt / 彼以熊来，吾以熊往
主题标签：朋友;敌人;边界
李敖引魏廷朝的话说，自己是最可靠的朋友，也是最难缠的敌人。人生里的善良不必等于软弱，重情也不必等于好欺负；对朋友可靠、对敌人难缠，才是一种完整的边界。

【SJIAN078】私事流言要交给时间和证物
来源章节：041.彼以熊来，吾以熊往.txt / 彼以熊来，吾以熊往
主题标签：流言;记录;事实
李敖在信中列举母亲、胡家等流言的来龙去脉，附记又用后来母亲回台同住证明“被赶出去”之说是造谣。私事流言常越解释越乱，但时间、文件和后来事实会留下证词；把记录留好，迟早能让谣言露出底。

【SJIAN079】藏头露尾，不等于逃避
来源章节：042.神龙见尾不见首.txt / 神龙见尾不见首
主题标签：隐居;专注;力量
李敖解释战斗性隐居，说真正的神龙都是把头藏起来，藏头露尾于人足矣；再藏头也不是鸵鸟。专注大事时，减少露面不是怯懦，而是保存能量；关键看你是在躲责任，还是在蓄力量。

【SJIAN080】别让赴约打断宇宙大道理
来源章节：042.神龙见尾不见首.txt / 神龙见尾不见首
主题标签：专注;社交;工作
李敖讲牛顿回家拿酒却因灵感来了忘记朋友的故事，自嘲自己为了防止被饭局打断，干脆免下馆子。真正要做深工作的人，必须保护注意力；不是轻慢朋友，而是知道有些灵感和工程，经不起频繁社交切割。

【SJIAN081】处变时，要能放下笔杆谋生
来源章节：043.李氏理想，张氏实行.txt / 李氏理想，张氏实行
主题标签：谋生;尊严;处变
李敖回忆自己曾想卖牛肉面，张俊宏后来真开小吃店，意在证明知识分子离开笔杆、讲坛也能谋生，不必仰人鼻息。一个人的尊严不该只绑在熟悉职业上；处境变了，能换一种方式活下去，才是真本事。

【SJIAN082】理想失败，也能证明现实的压力
来源章节：043.李氏理想，张氏实行.txt / 李氏理想，张氏实行
主题标签：失败;环境;理想
张俊宏的“相见小吃”最终因政治骚扰、人心走避而失败，李敖认为自己当年若开店，下场只会更坏。实验失败不一定说明理想荒唐，有时正说明外部压力有多重；看失败，要分清能力不足和环境围堵。
