【SZJ001】长期守规矩，是反击找麻烦的底气
来源章节：001.致管区警察头子书.txt / 致管区警察头子书
主题标签：规则;记录;自保
李敖先说明自己十七年来不找“小警察”的麻烦，户口校正等手续也照规矩办理，所以对方突然刁难时，他能有理有据地反击。做人不必处处争强，但平日守住记录、规矩和分寸，到了真正被找麻烦时，才有资格把话说重、把证据摆硬。

【SZJ002】对改错的人要留出夸奖空间
来源章节：001.致管区警察头子书.txt / 致管区警察头子书
主题标签：改错;宽严;分寸
管区后来承认误会并补办手续，李敖在跋里说公务员知过能改时应该夸奖。争理不是为了把人逼到永远下不了台，而是为了让事情回到正轨；对愿意改正的人保留肯定，能让原则不变成报复，也让下一次改错更容易发生。

【SZJ003】思想遗产要早做制度安排
来源章节：002.讨老婆的一个新害处.txt / 讨老婆的一个新害处
主题标签：传承;安排;思想
李敖从胡适、殷海光、雷震等例子看到，思想家的遗稿和著作若只靠家属情绪处理，很容易被拖延、封存或误解。重要工作不能等人死后才靠好意碰运气；越是关乎思想、资料和记忆的事，越要在生前安排清楚授权、保管和出版路径。

【SZJ004】别让身后加工改写生前主调
来源章节：002.讨老婆的一个新害处.txt / 讨老婆的一个新害处
主题标签：纪念;人格;传承
李敖批评殷海光死后被“党化”“国化”“神化”，主张认识他要看健康时期的理智表现和斗士性格。一个人身后常被亲友、组织或仪式改造成可利用的符号；真正尊重，是守住他生前最清醒、最锋利的主调，不让纪念变成改写。

【SZJ005】爱斗士，不要把他劝成普通人
来源章节：002.讨老婆的一个新害处.txt / 讨老婆的一个新害处
主题标签：使命;爱护;锋芒
李敖说殷海光和雷震都是斗士型人物，劝斗士不要斗，动机虽好，却可能是爱之适足以害之。爱一个人不是替他拿掉全部锋芒，而是理解他靠什么成为他自己；若把一个人的使命感劝没了，表面是保护，实则是削弱。

【SZJ006】结盟先问对象能否同路
来源章节：003.茅坑里的石头与厕所里的花瓶.txt / 茅坑里的石头与厕所里的花瓶
主题标签：结盟;判断;边界
李敖批评党外人士邀请附庸性政党共识座谈，提醒统战也要选择对象。合作不是人数越多越好，关键要看对象的真实立场、历史关系和可共同承担的风险；为了热闹而拉人入局，最后会把路线也弄混。

【SZJ007】识人要翻旧账，不要只听新口号
来源章节：003.茅坑里的石头与厕所里的花瓶.txt / 茅坑里的石头与厕所里的花瓶
主题标签：识人;历史;判断
面对青年党人谈“四立”、谈修养，李敖追溯他们过去两头做花瓶、内斗内行的历史。识人不能只看眼前漂亮口号，也要翻旧账、看长线行为；一个团体若长期在关键处站不稳，临时喊出的原则未必可靠。

【SZJ008】掌声会削弱面对真实的能力
来源章节：004.假戏真做与假戏假做.txt / 假戏真做与假戏假做
主题标签：真实;掌声;韧性
李敖由索菲亚罗兰坐牢后的恐惧想到，长期被奉承和捧场包围的人，容易退化面对真实人生的能力。一个人若总活在掌声里，遇到一点真实痛苦就可能失衡；保持与真实生活的接触，比维持被欣赏的形象更重要。

【SZJ009】艰难日子也要先找一点开心
来源章节：009.志在搅局.txt / 志在搅局
主题标签：趣味;韧性;工作
李敖说早上起来先做一件开心的事，找出牢里搜集的漫画，写信给朋友。人在高压和劳累中，不必把自己压成纯苦行；先给一天一点轻松和兴味，反而能让后面的工作与战斗继续下去。

【SZJ010】不见面也可以换一种交朋友方式
来源章节：005.我们要有批评党外的自由.txt / 我们要有批评党外的自由
主题标签：友情;距离;方式
李敖说自己隐居并不是同好朋友绝交，只是换成纯写信的交友方式；若以为不见面就不是朋友，是太感情型。关系不一定靠频繁相聚证明，有时换一种节奏和媒介，反而能让彼此保留空间、减少消耗、继续有质量地往来。

【SZJ011】自己阵营也要有被批评的自由
来源章节：005.我们要有批评党外的自由.txt / 我们要有批评党外的自由
主题标签：批评;阵营;自由
李敖坚持不但要有批评国民党的自由，也要有批评党外的自由；党外若不许人批评，心态和手法就与国民党无异。真正的共同体不能只欢迎对外攻击，也要容纳内部纠错；越说自己代表自由，越要先证明自己承受批评。

【SZJ012】形式在场不等于真实代表
来源章节：006.论“影子政权”.txt / 论“影子政权”
主题标签：判断;正当性;独立
李敖借“我在，故我思”转到政治判断，认为有些政权虽口口声声代表别人，其实只是影子。人生中也常有类似处境：某些关系、职位或权威只是形式在场，却没有真实代表性；判断它们时，要看实质承担，而不是看名义存在。

【SZJ013】公私不分会毁掉读书人的体面
来源章节：007.学者名流与公帑.txt / 学者名流与公帑
主题标签：公私;资源;操守
李敖批评钱穆、林语堂等以公帑建馆归隐，认为私人归隐不能霸占公产。一个人越讲仁义道德，越要在钱、资源和权力上守住公私界线；读书人的体面不是说出来的，而是在利益面前分得清公与私。

【SZJ014】渐老时更该做非自己不可的事
来源章节：011.国民党与豹.txt / 国民党与豹
主题标签：使命;选择;晚年
李敖劝刘绍唐，大家都渐渐老去，总该做点别人做不成、非我们做不可的事，并且不避嫌疑与困难。人生越往后，越要把精力放在自己独有的位置上；别人也能做的事可以让路，非你不可的事才值得扛起来。

【SZJ015】改革不能停在抽象注意事项
来源章节：008.给黄少谷先生的又一公开信.txt / 给黄少谷先生的又一公开信
主题标签：改革;案例;制度
李敖说许多“不能断章取义”“不能以推测入罪”的规定早已存在，若法官听者藐藐，抽象注意事项仍然无效。改革不能只增加漂亮条文，还要用具体案例做机会教育；只有让错误在实例中暴露，制度才知道该改在哪里。

【SZJ016】法律判断不能脱离实际作业
来源章节：008.给黄少谷先生的又一公开信.txt / 给黄少谷先生的又一公开信

李敖批评法官不了解银行作业，误以为抵押期限未到就无拍卖之虞，因而把保全房屋的行为判成犯罪。判断一件事不能只套抽象概念，还要理解真实流程；不懂业务细节的裁断，越自信越可能伤人。

【SZJ017】冤案平反虽晚仍要拿回
来源章节：008.给黄少谷先生的又一公开信.txt / 给黄少谷先生的又一公开信

最高法院判决使李敖坐过的冤狱获得平反，他承认西方谚语说迟来的正义不算正义，但对已受害者而言仍愿意算。受了冤屈不要因为时间已晚就放弃追索；迟来的平反不能抹掉伤害，却能把被夺走的名分和记录拿回一部分。

【SZJ018】把多年见闻结成自己的证词

李敖建议高信疆写一部回忆录，把报阀内幕和儒林外史揭发出来，也算对多年见闻做一总结。一个人在体制里待过，不能只带走委屈和资历；把看见的结构、经验和内幕整理成证词，才可能把被动经历变成主动资产。

【SZJ019】一个阶段完成后要敢退出来

【SZJ020】被上一代压久了，也要学会翻掌自立

【SZJ021】史料工作不能只给旧势力自我陶醉

李敖肯定《传记文学》结合史料的努力，也批评它让许多旧人物在杂志上自我陶醉、延年益寿。保存材料不能变成替旧势力擦粉；真正的史料工作要让读者从缝隙中看见真相，而不只是给当事人留下漂亮回忆。

【SZJ022】不要把可贵平台用来写无关痛痒
来源章节：011.国民党与豹.txt / 国民党与豹

李敖批评何凡独占专栏多年，却没把言论自由尺度写宽一点，只写琐事。拥有持续表达的位置，是难得的公共资源；若只用它保持安全和体面，而不推动边界，平台越大，失职也越大。

【SZJ023】只骂别人不许被骂不是自由

李敖说自由民主就是“人骂我我也骂人”，只骂人不许人骂，不是自由民主，而是秦始皇。真正相信自由的人，不能只把它当成攻击别人的武器，也要让别人拿同一把尺量自己；不能承受回骂，就别高谈开放。

【SZJ024】有勇气办事就别急着切割风险

《前进周刊》在编者按里写文责由李敖个人承担，李敖认为办杂志者不应显得临难苟免。合作做有风险的事，当然要分清责任，但不能一出事就先把自己摘干净；真正的担当，是知道风险仍愿共同承担。

【SZJ025】问法条就答法条，不要用口才逃题

苏秋镇追问禁止办报根据哪条法律，宋楚瑜和孙运璿答了半天却讲别的，李敖据此指出他们不依法办事。面对具体问题，最坏的聪明是用口才绕开题目；真正负责的回答，要正对问题，而不是把流利当作答案。

【SZJ026】理由换了，也要公开交代

【SZJ034】爱人以德，不要姑息他犯错

【SZJ035】批评样板不等于私人恩怨

【SZJ037】拥护原则，不要惯性拥护人
来源章节：019.几个主题的讨论.txt / 几个主题的讨论
