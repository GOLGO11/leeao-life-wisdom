【WCE001】刑罚先受法理约束
来源章节：001.“缇萦救父”表示了什么？.txt / “缇萦救父”表示了什么？
主题标签：法治;刑罚;身体
刑罚如果只是权力先抓人、先伤身，再替自己补上法律名目，就不是正义而是整人技术。判断制度文明程度，不能只看惩罚有没有条文，更要看它是否把人的身体、自由和程序当回事；法理管不住的刑罚，穿上法律外衣也仍是暴力。

【WCE002】技术进步不等于文明进步
来源章节：001.“缇萦救父”表示了什么？.txt / “缇萦救父”表示了什么？
主题标签：技术;文明;暴力
现代手段可以把折磨做得更隐蔽、更科学、更无痕，但这不是文明，而是暴力升级。一个社会越会使用技术，越要警惕技术被拿来消灭证据、延长痛苦、包装残忍；真正的进步，是减少施暴的权力，而不是改良施暴的方法。

【WCE003】德政要看落地后的代价
来源章节：001.“缇萦救父”表示了什么？.txt / “缇萦救父”表示了什么？
主题标签：改革;制度;代价
统治者一句废除恶法，若换来的只是更重的替代惩罚，慈悲就可能变成漂亮招牌。看一项改革，不要只听它宣布时多仁厚，要追问执行后谁受益、谁受苦、旧恶是否换了名字继续存在。

【WCE004】别让家族吞掉个人
来源章节：001.“缇萦救父”表示了什么？.txt / “缇萦救父”表示了什么？
主题标签：个人;家族;责任
以家族为单位判断罪责和荣辱，看似重情，其实最容易让无辜者替别人承受后果。成熟的人伦和法律，都要先承认每个人有自己的边界；亲人可以相爱、相救，却不能因此被制度合并成同一个责任主体。

【WCE005】善良不能替制度补漏洞
来源章节：001.“缇萦救父”表示了什么？.txt / “缇萦救父”表示了什么？
主题标签：善良;制度;代价
愿意替亲人受苦当然动人，但正因为动人，制度更不该把这种善良当成解决方案。一个社会若需要好人卖掉自己来弥补刑罚残酷，问题不在善良不够，而在制度让善良承担了不该承担的成本。

【WCE006】人格权不能拿来交易
来源章节：001.“缇萦救父”表示了什么？.txt / “缇萦救父”表示了什么？
主题标签：人格权;自由;尊严
人的自由、身体、姓名、名誉这些最基本的东西，不能像财产一样买卖、继承或抵押。一个人若被迫用自己的人格去换别人的安全，故事再感人，也说明社会还没有把人真正当人。

【WCE007】孝道故事先看被牺牲的人
来源章节：001.“缇萦救父”表示了什么？.txt / “缇萦救父”表示了什么？
主题标签：孝道;牺牲;尊严
传统常把牺牲包装成孝行，却少问牺牲者后来怎样活。读这类故事，不能只被感动带走，还要看被颂扬的人是否失去自由、尊严和选择；没有人的完整，孝道就会变成漂亮的伤口。

【WCE008】特权不是平权
来源章节：002.给杨贵妃的一封信.txt / 给杨贵妃的一封信
主题标签：女性;平权;特权
一个女人因美貌被帝王宠爱，可能暂时抬高某些女人的身价，却不等于女性真的拥有权利。个别人的得宠只是特例，结构中的平等才是权利；把少数特权误认为群体解放，是最容易上当的乐观。

【WCE009】危机中最弱者常被推成替罪羊
来源章节：002.给杨贵妃的一封信.txt / 给杨贵妃的一封信
主题标签：责任;替罪羊;弱者
盛世里被赞美的人，乱局一来就可能被推出去背锅。看一场失败，不要急着寻找最方便牺牲的人，而要追问权力、制度和决策者怎样把责任转嫁给不能自辩者。

【WCE010】一字之差会改写判断
来源章节：003.“逸豫适足亡身”吗？.txt / “逸豫适足亡身”吗？
主题标签：逻辑;语言;判断
把“可以”偷换成“必然”，格言就会从提醒变成恐吓。人生判断常坏在这种细小偷换上；越是熟悉的名句，越要分清它说的是可能、条件还是结论，否则一句话就会把人带进错误。

【WCE011】格言离开语境就会变棍子
来源章节：003.“逸豫适足亡身”吗？.txt / “逸豫适足亡身”吗？
主题标签：语境;格言;判断
古人的话不是随手可挥的棍子，先要回到原文、对象和处境。任何道理都有它成立的边界；不问语境，只用名言压人，表面像有学问，实际上是在偷懒。

【WCE012】好问题不能先逼人认输
来源章节：003.“逸豫适足亡身”吗？.txt / “逸豫适足亡身”吗？
主题标签：问题;思考;自由
一个题目若先把结论藏在题干里，回答者就不再是在思考，而是在替题目辩护。无论考试、讨论还是自我反省，真正好的问题都应该给人判断空间；问题设坏了，答案再漂亮也不自由。

【WCE013】舒适不该天然有罪
来源章节：003.“逸豫适足亡身”吗？.txt / “逸豫适足亡身”吗？
主题标签：生活;舒适;清醒
安乐可能腐蚀人，也可能只是普通人应得的体面生活。不要用艰苦神话恐吓一切舒适，也不要把享受生活一概说成堕落；真正要问的是人有没有因此失去清醒、责任和创造力。

【WCE014】别把所有事都拖去大义名下
来源章节：003.“逸豫适足亡身”吗？.txt / “逸豫适足亡身”吗？
主题标签：大义;现实;尺度
什么都往报国、主义、宏大使命上扯，会让现实问题变得模糊。人生需要大方向，也需要承认具体事有具体尺度；小事先按小事处理，世界反而更清楚。

【WCE015】事物自身也有价值
来源章节：004.中国小姐新论.txt / 中国小姐新论
主题标签：审美;价值;尺度
美可以作为美被看见，不必先被政治、道德或宣传征用。许多事情都有自己的尺度，尊重它，才不会把世界简化成工具箱；凡事都要挂上大旗，反而失去对事物本身的感受力。

【WCE016】禁止和开放都可能同样粗暴
来源章节：004.中国小姐新论.txt / 中国小姐新论
主题标签：判断;道德;政治
以道德名义禁止，和以政治名义开放，看似相反，实际都可能不尊重对象本身。判断一件事，不只问赞成还是反对，还要问你的理由是不是把它当成独立的事物，而不是当成口号材料。

【WCE017】公共命名是在安排记忆
来源章节：005.罗斯福路该改名罗斯祸路.txt / 罗斯福路该改名罗斯祸路
主题标签：命名;公共记忆;历史
路名、书名、奖名都不是小标签，它们会告诉后来的人该记住谁、尊敬谁、忘掉谁。给公共事物命名时，先要问价值排序，而不是只看一时方便；名字一旦留在空间里，就会替历史说话。

【WCE018】改名不是泄愤要有理由
来源章节：005.罗斯福路该改名罗斯祸路.txt / 罗斯福路该改名罗斯祸路
主题标签：改名;理性;历史
把名字改掉很容易，难的是说明为什么改、按什么标准改、改后要记住什么。若只是把情绪刻到招牌上，改名就会变成另一种宣传；真正负责任的改名，要经得起历史和公共理性的追问。

【WCE019】重要性常在处境里
来源章节：006.重要不重要与不重要重要.txt / 重要不重要与不重要重要
主题标签：处境;重要性;理解
一件事对旁人像小泥块、小扫帚，对当事人却可能关乎生活秩序。不要太快替别人判定“不重要”；先问它在谁的处境里发生、牵动什么后果，人的理解才会变得具体。

【WCE020】别用更重要来封口
来源章节：006.重要不重要与不重要重要.txt / 重要不重要与不重要重要
主题标签：公共讨论;问题;认真
世界上永远有更大的问题，但这不能成为不谈眼前问题的借口。公共讨论和日常处事都一样，不能拿大题目压死小题目；能把一个个具体问题说清、办好，才是认真。

【WCE021】小问题也要一个萝卜一个坑
来源章节：006.重要不重要与不重要重要.txt / 重要不重要与不重要重要
主题标签：行动;具体问题;方法
事情再多，也不是靠一句“都很复杂”就能解决。把问题逐个放回自己的位置，能办一个算一个，社会和人生才会往前动；宏大抱怨不如具体处理。

【WCE022】发问者不必替执行者沉默
来源章节：006.重要不重要与不重要重要.txt / 重要不重要与不重要重要
主题标签：发问;职责;公共生活
舆论、朋友、旁观者的责任是把问题说出来，掌权者和当事者的责任是想办法做。不能因为自己不是执行者就闭嘴，也不能因为别人多言就恼怒；各守其职，事情才有被看见和改正的机会。

【WCE023】法律判断先看权利和事实
来源章节：007.张小兰冤在哪儿？.txt / 张小兰冤在哪儿？
主题标签：法律;事实;权利
一个案件若被道德议论、身份成见和动机猜测淹没，受伤最深的是事实本身。判断争端先问权利是否被侵犯、证据是否成立，再谈其他评价；顺序乱了，公道就会走样。

【WCE024】别先审问受害者的人格
来源章节：007.张小兰冤在哪儿？.txt / 张小兰冤在哪儿？
主题标签：正义;受害者;偏见
人是否值得保护，不取决于他在旁人眼里够不够完美。若每个求助者都先被查性格、职业、名声和动机，正义就会变成少数“干净人”的特权。

【WCE025】动机猜测不能代替证据
来源章节：007.张小兰冤在哪儿？.txt / 张小兰冤在哪儿？
主题标签：证据;动机;判断
怀疑别人为什么这样做，很容易；证明事情到底怎样发生，却需要证据。诛心之论最省力，也最容易制造冤屈；成熟判断要把“我猜他想什么”和“我知道发生了什么”分开。

【WCE026】解释要容得下多义
来源章节：008.“舒而脱脱兮！”.txt / “舒而脱脱兮！”
主题标签：解释;多义;判断
经典、诗句和人的行为，常常不止一种解释。承认多义不是放弃判断，而是避免把自己的看法冒充唯一真理；越复杂的对象，越需要给解释留下余地。

【WCE027】道德标签会偷走复杂性
来源章节：008.“舒而脱脱兮！”.txt / “舒而脱脱兮！”
主题标签：道德标签;复杂性;判断
一个标签贴上去，旁人就以为事情已经判断完了。许多现象真正值得分析的部分，正是在标签遮住的地方；先把标签拿远一点，才看得见动机、技术、处境和边界。

【WCE028】轻视之前先理解专业
来源章节：008.“舒而脱脱兮！”.txt / “舒而脱脱兮！”
主题标签：专业;理解;评价
哪怕是被社会看低的行业，也可能有训练、规矩和熟练技术。轻蔑常来自无知；先弄清楚一件事怎样做、难在哪里，再决定如何评价，判断才不会廉价。

【WCE029】压制有影响者常会放大他
来源章节：009.过早的答案.txt / 过早的答案
主题标签：压制;反作用;影响力
对一个已有影响的人下重手，未必能让他消失，反而可能替他制造更大的象征性。处理冲突要会计算反作用；有时最笨的伤害，是把对方推上更高的位置。

【WCE030】要求权利不是大逆不道
来源章节：009.过早的答案.txt / 过早的答案
主题标签：权利;公民;自尊
一个公民要求工作、表达和基本保障，不是在闹事，而是在把应有的东西说出来。人不要被权力或习俗吓到，把自己的正当权利说成恩赐；先承认它本来属于人，才可能守住它。

【WCE031】不要迷信政治能领导一切
来源章节：009.过早的答案.txt / 过早的答案
主题标签：政治;思想;改良
政治重要，但不是所有改变的唯一入口。若把人生和社会的希望全押在权力转换上，就容易忽略教育、思想、习惯、行业和具体生活的改良；许多深变，是从权力之外慢慢长出来的。

【WCE032】思想变化是长效动力
来源章节：009.过早的答案.txt / 过早的答案
主题标签：思想变化;判断力;成长
真正稳定的改变，不只是换位置、换口号，而是人们看事情的方式变了。一个人要改变自己，也不能只换环境和工具，还要让判断力、价值排序和常识一起更新。

【WCE033】实际改良比空喊更爱人
来源章节：009.过早的答案.txt / 过早的答案
主题标签：改良;实事;责任
关心国家或他人，不一定表现为高谈权力大局，也可以是研究农村、改善小民处境、督促恶法修正。爱若没有落到具体问题上，很容易变成漂亮口号；能解决一两件实事，才算有重量。

【WCE034】高论要有可实行的出口

主题标签：可行性;行动;判断

【WCE035】两边夹攻未必说明你错

【WCE036】别把小聪明用来害人

主题标签：小聪明;责任;名誉
借别人的手、别人的名义、别的制度去整人，看似聪明，其实要承担更深的名誉成本。处世最危险的聪明，是以为自己躲在背后就没有责任；伤害一旦被看穿，坏名声会回到操盘者身上。

【WCE037】给后来者让路也是智慧
来源章节：009.过早的答案.txt / 过早的答案
主题标签：让路;更新;时代
旧人物若只会挡路，会把自己从前的地位变成笑柄。一个人到了该退、该让、该更新的时候，要有把路让开的气度；时代不会因为谁恋栈就停下来。

【WCE038】背信时先把事实固定下来
来源章节：010.一封存证信的故事.txt / 一封存证信的故事
主题标签：背信;文件;追责
对方收了钱、给了收据、又突然翻脸，这时愤怒不够，文件才有用。遇到失信，先把时间、凭证、承诺和损失固定下来；能被核对的事实，是追责的起点。

【WCE039】公道要落实在让对方说话
来源章节：010.一封存证信的故事.txt / 一封存证信的故事
主题标签：公平;回应权;言论
一边刊登攻击别人的话，一边拒绝当事人的回应，再高喊公道也站不住。判断一个人或机构是否公平，不听口号，先看它是否允许被批评者同样发声。

【WCE040】不要迷恋悲壮牺牲

主题标签：牺牲;理想;生命
为理想牺牲固然震动人，但把流血当成必要条件，很容易把年轻生命送给旧制度。能保存力量、等待机会、继续做事，未必不如慷慨赴死；人生不能把悲壮当成唯一的高贵。

【WCE041】最后姿态不能全由自己掌握
来源章节：011.论没有“流血的自由”.txt / 论没有“流血的自由”
主题标签：抗议;证据;作品
人常以为临终一刻可以说出真心、完成抗议，但更狡猾的压迫会先夺走发声机会和信用。不要把全部意义寄托在最后表演上，平日留下证据、朋友和作品，才更可靠。

【WCE042】毁人之前常先毁信用
来源章节：011.论没有“流血的自由”.txt / 论没有“流血的自由”
主题标签：信用;污名;自保
现代压迫未必先动手，它会先破坏一个人的历史、名誉、发表渠道和关系网络。保护自己和别人，不能只等最后危机，还要在信用被涂黑时就警觉、澄清和记录。

【WCE043】评断屈辱沉默要看牵制

【WCE044】公开会改变伤害的成本

【WCE052】批评先看指涉点
