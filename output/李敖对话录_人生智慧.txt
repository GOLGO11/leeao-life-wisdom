【DHL001】对话要能质疑和抬杠
来源章节：《李敖对话录》自序.txt / 《李敖对话录》自序
主题标签：对话;质疑;思想
对话的价值不在互相点头，而在用通俗的口语把质疑、问难和抬杠摆出来。思想若只能独白，容易自满；能经得起对话，就多了一层被追问、被修正、被照亮的机会。

【DHL002】权力要受思想牵引
来源章节：001.才华盖世侠骨柔情的思想家.txt / 才华盖世侠骨柔情的思想家
主题标签：政治;思想;方向
权力若只由掌权者自己转圈，水准很容易降成权术和经营。真正能提升公共生活的，是让思想、批评和原则持续提出方向；权力需要被思想牵引，才不至于只剩自我扩张。

【DHL003】活得痛快也要有冷静底盘
来源章节：001.才华盖世侠骨柔情的思想家.txt / 才华盖世侠骨柔情的思想家
主题标签：个性;理智;表达
痛快不是胡来，狂放也不是放弃思考。一个人可以表达得锋利、戏剧化，甚至带一点撒野，但底下要有长期读书、独立思考和冷静判断作支撑；没有底盘的痛快，只是噪音。

【DHL004】群众面前更不能讨好群众
来源章节：002.对是非绝对是不让步的.txt / 对是非绝对是不让步的
主题标签：群众;是非;独立
群众会欢呼，也会看热闹；会抬人，也会毁人。面对群众时，最危险的是为了掌声而放弃是非。真正有担当的人，不是顺着群众情绪讨好，而是在群众面前仍坚持判断。

【DHL005】孤独先要调整心境
来源章节：002.对是非绝对是不让步的.txt / 对是非绝对是不让步的
主题标签：孤独;心境;独立
人在孤独无助时，最容易滑向自怜，把自己变成外界环境的函数。真正的独立作战，是先调整心境，把生活秩序握在自己手里；外界可以隔绝你，但不能替你决定内心怎样运转。

【DHL006】苦境中还能工作才见硬度
来源章节：002.对是非绝对是不让步的.txt / 对是非绝对是不让步的
主题标签：苦境;纪律;韧性
环境艰苦本身不能证明一个人强，能在艰苦中保持情绪不坏、生活有序、工作不断，才是真硬度。痛苦若只把人带向怨怼，就白受了；能把痛苦转成纪律，才算真正撑住。

【DHL007】专业身份不能替代现实勇气
来源章节：002.对是非绝对是不让步的.txt / 对是非绝对是不让步的
主题标签：知识分子;勇气;现实
专业成就不能抵消现实逃避。一个人若只在座谈会里表演一页心得，却不敢批评畸形现象、不肯深入问题现场，就是把学问变成装饰。真正的学问，应当有进入现实的勇气。

【DHL008】民主看有没有开玩笑的自由
来源章节：002.对是非绝对是不让步的.txt / 对是非绝对是不让步的
主题标签：民主;幽默;自由
民主不只在宪法、议会和选举里，也在日常空气里。一个地方若上下都能互相开玩笑，权力能被幽默触碰，人们能轻松表达不敬，民主才可能生活化；不能开玩笑的民主，多半只是招牌。

【DHL009】思想要看日常行为
来源章节：002.对是非绝对是不让步的.txt / 对是非绝对是不让步的
主题标签：思想;日常;观察
理解一种思想，不能只盯着经典名家和上层论述，也要看普通人怎样祭拜、服从、抗拒、说话和选择。真正影响社会的思想，往往藏在男女老幼的日常行为里。

【DHL010】坏榜样也能教育你
来源章节：003.在台大·在法庭.txt / 在台大·在法庭
主题标签：教育;自学;反面教材
学校和老师不一定都能正面成全人，有时他们的呆板、老化和窄路，反而提醒你不要变成那样。负面作用若被看清，也会变成一种教育：它逼你另找宽广的路。

【DHL011】独立思考需要自己的时间
来源章节：003.在台大·在法庭.txt / 在台大·在法庭
主题标签：独立思考;时间;读书
独立见解不是凭姿态生出来的，它需要长时间和别人隔开，反复读书、思考、推翻、再确认。能忍受一段不热闹的时间，才可能走过“见山不是山”，再回到自己的“见山是山”。

【DHL012】争不到表决权也要争发言权
来源章节：004.吐他一口痰.txt / 吐他一口痰
主题标签：发言权;抗议;限度
在不公平的结构里，不一定马上争得到最后决定权，但仍要争取发言权。能公开说不服、能杯葛、能抗议，本身就是位置；若连表达都放弃，就只剩被安排的角色。

【DHL013】情绪要看过滤后剩下什么
来源章节：004.吐他一口痰.txt / 吐他一口痰
主题标签：表达;情绪;内容
表达有情绪并不可怕，关键是把情绪过滤掉以后，还剩不剩见解、资料和事实。冷冰冰的错误仍是错误，火辣辣的正确仍可能有价值；判断文章，不要只看声调，要看内容。

【DHL014】语言要有个人锋芒
来源章节：004.吐他一口痰.txt / 吐他一口痰
主题标签：语言;表达;个性
官样话最坏的地方，是把不同的人磨成同一种腔调。真正有生命的表达，应当显出个人锋芒、立场和节奏；如果甲乙丙丁说出来全无差别，语言就已经替思想投降了。

【DHL015】羊圈里学不成老虎
来源章节：005.答陈依玫.txt / 答陈依玫
主题标签：教育;自学;个性
制度化教育若只把人训练成温顺的羊，就不适合所有个性和才气。想成为老虎，不能只等学校批准你长牙，还要自己找书、找路、找训练；文凭可能有用，但不该决定你的边界。

【DHL016】失败者的王牌是不怕失败
来源章节：005.答陈依玫.txt / 答陈依玫
主题标签：失败;勇气;行动
一个人若已经看清失败的可能，就不容易再被失败吓倒。失败者最有力量的一张牌，是不再把失败当成终极恐吓；还能继续写、继续做、继续过江，失败就只是路况，不是结论。

【DHL017】好文章要有话直说、小心求证
来源章节：005.答陈依玫.txt / 答陈依玫
主题标签：写作;白话文;求证
理想的白话文，不必装腔作势，也不能只凭痛快乱打。好的文字应该有话直说、有屁直放，同时小心求证、当文不让；直率和严谨合在一起，才是有力的表达。

【DHL018】接棒不能靠人事和马屁
来源章节：005.答陈依玫.txt / 答陈依玫
主题标签：接棒;能力;人格
真正的接棒人，不该靠关系、马屁和安排被接上去。靠人事得来的棒，容易误己也误事；能接住未来的人，必须有目标、能力、悍气和独立承担的准备。

【DHL019】众口一声时要有人唱反调
来源章节：006.论国民党的美感.txt / 论国民党的美感
主题标签：异议;知识分子;真理
在众口一声的环境里，反对、批评和异议不是添乱，而是求真的必要位置。真理常从反面被看见，也需要第二个、第三个声音接力支持；没人唱反调，错误就会装成共识。

【DHL020】概念不能随便加帽子
来源章节：006.论国民党的美感.txt / 论国民党的美感
主题标签：概念;标准;判断
有些概念本身就有清楚标准，不能随便加上地方性、权宜性或宣传性的帽子来偷换。民主就是民主，美感就是美感；一旦什么都可另加解释，概念就被权力和庸俗抹花了。

【DHL021】思想家要提出并解决痛苦问题
来源章节：007.我的皮肉生涯.txt / 我的皮肉生涯
主题标签：思想家;问题意识;时代
乱世里最缺的不是情绪，而是能把痛苦问题提出、研究、并设法解决的人。思想家不能只会喊口号，也不能满足于做政客的注脚；他要困学知变，替时代找出真正的问题。

【DHL022】超前的人常难在生前被接受
来源章节：007.我的皮肉生涯.txt / 我的皮肉生涯
主题标签：先知;孤独;时代
先知式的人物难在有生之年被人接受，因为他常常比时代快一步。若你选择走在前面，就要预备承受误解和孤独；被当时接受不一定是真成就，不被接受也不必然是失败。

【DHL023】关系不是即溶咖啡
来源章节：008.党外人士不够小.txt / 党外人士不够小
主题标签：关系;细节;诚意
人与人的关系靠持续的细心、关切、照顾和礼貌，不是有求于人时热情，用完就消失。所谓“小”功夫，其实是诚意的日常形态；让人感到被尊重，比临时表演更有力量。

【DHL024】小功夫要服从大方向
来源章节：008.党外人士不够小.txt / 党外人士不够小
主题标签：细节;大方向;诚意
细节做得好很重要，但若大方向错，细节就会变成权谋术数。小功夫只有建立在正确方向和真诚基础上，才是感人；脱离方向的周到，只是精致的算计。

【DHL025】参与坏游戏前先看是否成背书
来源章节：009.选举问题总解答.txt / 选举问题总解答
主题标签：体制;参与;清醒
在不公平的规则里，参与未必等于改变，有时反而替体制制造正常运作的假象。进入任何游戏前，都要看清规则由谁制定、你的角色是不是花瓶；别把被利用误认成有位置。

【DHL026】目的和手段要尽量一致
来源章节：009.选举问题总解答.txt / 选举问题总解答
主题标签：目的;手段;人格
为了反对坏东西而采用坏东西的规则，常会留下余憾。一个人若想保持光明正大，就要尽量让目的和手段一致；手段会塑造结果，也会悄悄塑造自己。

【DHL027】言论空间不是恩赐
来源章节：010.李敖答问之一.txt / 李敖答问之一
主题标签：言论自由;风险;制度
一个人能大胆说话，不一定证明统治者宽大，也可能只是他付出风险后争来的缝隙。评估自由，不能把少数人的不要命算成制度进步；自由要看普通人是否也能安全表达。

【DHL028】给权力打分数要查事实
来源章节：010.李敖答问之一.txt / 李敖答问之一
主题标签：事实;权力;判断
替权力说好话，最怕只凭印象和短期比较。真正的判断要查历史、证据和前后对照；若不看事实就给高分，那个分数只是权力想听的掌声。

【DHL029】争理之后还要承认自己可能过重
来源章节：011.李敖答问之二.txt / 李敖答问之二
主题标签：自省;分寸;关系
坚持是非不等于自己永远分寸刚好。对朋友、同道或后辈下手太重时，愿意接受提醒、承认过失，也是一种理性；争理之后还能修正自己的力度，才不是把原则变成脾气。

【DHL030】忠奸标准不要交给权力认证
来源章节：011.李敖答问之二.txt / 李敖答问之二
主题标签：评价;权力;标准
权力迟来的平反和褒扬，不一定是真公道，也可能是另一种收编。判断一个人的忠奸、抗议和精神，不该交给曾经压迫他的体制来检定；标准若交出去，纪念也会变成侮辱。

【DHL031】真诚能越过身份隔阂
来源章节：012.李敖答问之三.txt / 李敖答问之三
主题标签：真诚;身份;信任
语言、籍贯和标签不一定决定人与人的信任。一个人即使不懂某种地方话，只要长期以诚意待人，也可能赢得认同；真正打动人的，往往不是身份表演，而是至诚。

【DHL032】公关不能代替做事
来源章节：012.李敖答问之三.txt / 李敖答问之三
主题标签：公关;行动;贡献
一个人若只靠写信、拜寿、切蛋糕和到处露脸维持存在，终究是在用公关代替行动。真正的价值要看做了什么、承担了什么、战斗了什么；热络关系不能冒充实际贡献。

【DHL033】复杂著作要先点破限度
来源章节：013.“却笑英雄‘有’好手，一江春水走曹瞒”.txt / “却笑英雄‘有’好手，一江春水走曹瞒”
主题标签：读书;限度;判断
有些书材料丰富、推理细密，却可能在根本理论上误导人。读这种书，不能被学问架势吓倒，要先看它的限度在哪里；会利用也会警惕，才不至于被大书牵着走。

【DHL034】谈历史先辨史料真假
来源章节：013.“却笑英雄‘有’好手，一江春水走曹瞒”.txt / “却笑英雄‘有’好手，一江春水走曹瞒”
主题标签：历史;史料;专业
谈历史不能只靠热闹叙述和望文生义，先要辨别史料真假、概念含义和地名制度。基础没弄清，再流畅的解释也会变成误导；历史判断的第一步，是承认自己必须懂行。

【DHL035】小题目也能看见大世界
来源章节：013.“却笑英雄‘有’好手，一江春水走曹瞒”.txt / “却笑英雄‘有’好手，一江春水走曹瞒”
主题标签：研究;小题大作;历史
大问题未必要靠大题目来写。真正有功力的人，能从制度、人物或一段细小材料里看出整个时代；小题大作不是格局小，而是从一粒砂里看世界。

【DHL036】重要问题要个案详追
来源章节：014.台湾最有力量的个人.txt / 台湾最有力量的个人
主题标签：分析;个案;真相
谈问题若只点到为止，容易停在表皮功夫。真正的分析要做到个案详追：把事实、前因、后果和结构一层层追下去；粗糙的判断节省时间，却浪费真相。

【DHL037】言论不畅会让内容欲说还休
来源章节：014.台湾最有力量的个人.txt / 台湾最有力量的个人
主题标签：言论自由;写作;文化
没有言论自由，写作者就会欲说还休，读者也会逐渐失去兴趣，最后形成表达和阅读的恶性循环。文化要有活力，先要让话能说透；说不透，作品就难免萎缩。

【DHL038】批评别人低格调前先拿出替代品
来源章节：014.台湾最有力量的个人.txt / 台湾最有力量的个人
主题标签：批评;创造;宽容
看不惯别人的表达，不能只会声明和禁止。若嫌格调低，就拿出更高的作品来取代；自己不能创造，却急着审判别人，容易变成另一种假道学和不宽容。

【DHL039】真相靠辩证讨论逼近
来源章节：015.给他来个“Seven Down”！.txt / 给他来个“Seven Down”！
主题标签：真相;辩证;公开
错误难免发生，关键在能不能开放资料、允许更正、互相辩证。真相不是在封锁中自动出现的，它要靠公开争论和证据往返逼近；越遮盖，错误越容易被制造。

【DHL040】报导内幕还要正确和解释
来源章节：015.给他来个“Seven Down”！.txt / 给他来个“Seven Down”！
主题标签：写作;事实;解释
内幕本身不是深刻，只有报导没有解释，或解释得错误，都会让揭发变成噪音。公共写作至少要守住两个基础：事实要正确，解释要高明；否则越热闹，越容易帮倒忙。

【DHL041】公开资料也能翻出新解释
来源章节：015.给他来个“Seven Down”！.txt / 给他来个“Seven Down”！
主题标签：资料;解释;洞察
不必迷信内幕，公开资料若重新组合、细读和解释，也能逼出有力判断。真正的功夫不只是拿到秘密，而是把人人看得到的东西看出别人没看见的意义。

【DHL042】义愤要经过细腻分析
来源章节：015.给他来个“Seven Down”！.txt / 给他来个“Seven Down”！
主题标签：揭发;分析;准确
面对黑暗，不是越快越好，也不是越猛越好。真正有效的揭示，需要资料运用、历史了解、起承转合和准确处理；粗糙的义愤容易消耗公信力，细腻的分析才更难被打倒。

【DHL043】读书要用主题贯串群书
来源章节：016.五十而不知天命.txt / 五十而不知天命
主题标签：读书;方法;系统
读书多不等于会读书，死守一本接一本，可能只是在左摘右丢。有效的读法，是以问题和主题贯串群书，系统地看、分类地藏、随时调度；知识要能被调用，才算进了自己。

【DHL044】传统要开清单逐项检定
来源章节：016.五十而不知天命.txt / 五十而不知天命
主题标签：传统;检定;方法
谈传统不能笼统说全好或全坏，而要一项项开清单：哪些仍可用，哪些该放下，哪些只剩仪式。分得越细，越能解决问题；大词越空，越容易护短或乱砍。
