【SLXY001】短文也要言有物
来源章节：《世论新语》自序.txt / 《世论新语》自序
主题标签：表达;写作;言之有物
李敖在自序里说《世论新语》多是短篇，但短不是随便，仍要有话可说、有秩序可看。人生中的表达也一样，篇幅小、场合急、时间短，都不能成为空洞的借口；越是短促的发言，越要把判断磨尖，把废话削掉，让一句话承担它该承担的分量。

【SLXY002】漂亮文字不能替代思想
来源章节：《世论新语》自序.txt / 《世论新语》自序
主题标签：思想;文风;表达诚实
自序里谈到文胜于质、头巾气太重的问题，提醒人不要把文字的漂亮误当成思想的有力。一个人说话写作若只剩腔调，最多赢得一时好看；真正能留下来的，是能把经验、判断和事实组织起来的能力。

【SLXY003】用锋利的人要承担锋利的代价
来源章节：001.新专栏——疯老虎出场.txt / 新专栏——疯老虎出场
主题标签：用人;个性;代价意识
李敖借“疯老虎”说出一种用人常识：一个人的锐气若正是他可贵之处，就不能又想借他的锋芒，又要求他温顺如猫。人生里结交、合作、用人都要明白，选择有棱角的人，等于选择了他的力量和代价；想要成果，就别幻想只保留好用的一面。

【SLXY004】没有原则的灵活只是互相利用
来源章节：002.高玉树与三面无耻.txt / 高玉树与三面无耻
主题标签：原则;机会主义;处世
文中把多方讨好、左右逢源看成三方面彼此利用的局面。灵活本来可以是智慧，但若没有价值底线，就会退化成谁都不真信、谁都可出卖的机会主义；成熟的人可以周旋，但不能把周旋当人格。

【SLXY005】离开组织也要问动机
来源章节：003.退党中的进步.txt / 退党中的进步
主题标签：动机;组织;自我审查
离开一个组织并不天然高尚，关键要看是为了原则、尊严和判断，还是为了利益、位置和风向。人生中退出、转向、换阵营都可能有正当理由，但最骗不了人的，是自己心里到底在追求自由，还是在追求更舒服的算盘。

【SLXY006】道德光环要接受利益盘点
来源章节：004.视听真乱了.txt / 视听真乱了
主题标签：清白;利益;名实检验
李敖质疑所谓高风亮节时，不停追问公产、特权和实际利益。一个人名声再好，也不能免于利益事实的盘点；真正的清白不怕问钱从哪里来、权从哪里来、方便从哪里来，怕追问的光环，往往只是灯影。

【SLXY007】批评远方以前先照见身边
来源章节：005.还是出击自己吧！.txt / 还是出击自己吧！
主题标签：自省;公共良知;责任
文中把对外部人权的关心拉回本地现实，提醒人不要只在安全距离外正义感充沛。人生里的批判若总挑远处发作、避开身边黑暗，就容易变成姿态；真正的良知，要敢从自己所在之地开始负责。

【SLXY008】欺负人之后再欺骗人更坏
来源章节：007.睁眼说瞎话.txt / 睁眼说瞎话
主题标签：诚实;谎言;尊重他人
李敖反复批评粗糙谎言，因为这种谎言不只遮盖事实，还把受众当傻瓜。做错事已经可恶，事后再用拙劣话术骗别人，就多了一层侮辱；真正的悔改从停止欺骗开始，而不是从包装错误开始。

【SLXY009】写作难在把责任接到底
来源章节：009.自己出版自己看.txt / 自己出版自己看
主题标签：写作;传播;承担
文中谈写作要有值得写的内容、敢出版的人，也要遇到懂得阅读的人。真正的写作者不能只怪环境，还要在找不到位置时自己承担传播责任；有些话若确实重要，就要想办法让它抵达，而不是只在抱怨中夭折。

【SLXY010】位置越高越不能说自己被蒙蔽

主题标签：领导责任;知情义务;担当
李敖嘲讽高位者总说被下面欺瞒，指出权力位置本身就包含知情和追责义务。一个人职位越高，越不能把无知当避难所；你可以被人骗一时，却不能长期靠“我不知道”来保住权威和清白。

【SLXY011】讨公道不能只讨自己人的公道
来源章节：014.“真心讨回历史公道”？.txt / “真心讨回历史公道”？
主题标签：公道;历史记忆;超越阵营
文中提醒人追索历史正义时，不可只看自己阵营的伤口。真正的公道有一种不舒服的要求：它会逼你看见自己人曾经亏欠过谁，也会要求你替不属于自己的人留下位置；只替同类喊冤，公道就会缩成私情。

【SLXY012】弱者面前最能验出自己的正义
来源章节：015.忘了我们是谁.txt / 忘了我们是谁
主题标签：弱者;正义;自我反省
李敖谈到原住民处境时，把被压迫者也拉进视野。一个人或一个群体若只记得自己受过苦，却看不见自己对更弱者的亏欠，正义感就还没有成熟；真正的觉醒，是在有机会占便宜时也愿意承认别人被亏待。

【SLXY013】改名词不等于改现实
来源章节：016.做戏无法，出个动词.txt / 做戏无法，出个动词
主题标签：名实;语言;改革
李敖对政治名词的包装保持怀疑，指出换一个好听说法并不会让问题消失。人生中许多改革也败在这里：把失败改叫转型，把管制改叫服务，把退让改叫务实；语言可以帮助理解，但不能替现实动手术。

【SLXY014】顺手骂人以前先查常识

主题标签：查证;批评;常识
文中多次抓出史实、地理、制度上的错误，提醒人愤怒不能免除查证义务。越是想批评别人，越要先把基础事实弄准；因为错误的批评不但伤不到对象，反而暴露自己的轻率，让正当立场也跟着失分。

【SLXY015】别用被盯上来证明一个人有罪
来源章节：018.施启扬未收红包？.txt / 施启扬未收红包？
主题标签：逻辑;证据;公平
李敖反对“为什么偏偏找你”式推理，因为这类推理先假定权力正确，再把被害处境当罪证。人生里判断他人，也要警惕这种偷懒逻辑；一个人被攻击、被检查、被传闻围住，并不自动说明他该受这一切。

【SLXY016】公布财产还要说明来源
来源章节：021.公布财产不够，报出来源才算.txt / 公布财产不够，报出来源才算
主题标签：财产;透明;来源意识
财产透明不只在于把数字摊出来，还要解释数字从何而来、是否合乎身份和收入。做人做事同样如此，交代结果容易，交代来路才难；真正经得起看的清白，不怕别人沿着来源一路问下去。

【SLXY017】要求别人守规矩时自己也要守规矩
来源章节：022.别再教猴子爬树了！.txt / 别再教猴子爬树了！
主题标签：规则;一致性;原则
李敖批评一边喊回归宪法、一边又要求总统越权的矛盾。人生中的原则最怕只拿来约束别人；若规则对你有利时才神圣、对你不便时就弹性，它就不再是原则，而只是换了名字的私心。

【SLXY018】坏人嘴里也可能有真话
来源章节：023.没有水平的台湾土蛋.txt / 没有水平的台湾土蛋
主题标签：不因人废言;判断;事实
文中强调不能因人废言。成熟的判断不是把人先分成好坏两堆，然后决定谁的话可听，而是把具体说法拿出来验；真话不会因为出自讨厌的人而变假，假话也不会因为出自喜欢的人而变真。

【SLXY019】别让小地方把心胸一起变小

李敖感叹人被小环境和小对手耗住之后，容易把人生比例也缩小。人在狭窄处境中更要保留大尺度，不然斗久了就会只会计较眼前恩怨；真正的自保，是不让环境替你规定灵魂大小。

文中对迟来的怀念和表态很不耐烦，因为真正的情义应当在对方还需要时出现。人生里许多漂亮纪念都输给一个及时动作；雪中一把力，胜过日后万字追思。

【SLXY021】理想一变成上班时间就危险了

李敖讽刺组织从理想变成办公室作息，提醒人警惕信念被职位化。真正的理想不能只在上班时间存在，也不能只在有预算、有名片、有舞台时存在；一旦下班就无信仰，所谓事业多半只是饭碗。

书中多处显示，信件、记录、公开文字可以让弱者不只凭口头喊冤。人生中遇到冲突，能把事实写清、时间排好、证据留下，就是一种力量；会记录的人，常比只会生气的人更能保住主动权。

文中讽刺那些热衷制定新规矩却不肯守现成规矩的人。规则的威信不是靠口号建立，而是靠每个人在不方便时仍然遵守；只爱用法律管别人、不爱用法律约束自己，最后会把法治讲成表演。

【SLXY028】纸上权利不能兑现会伤害真相信的人

李敖指出，制度若只在纸面许诺权利，却让认真相信的人吃亏，真正可笑的不是信法者，而是制度本身。人生里也要警惕空头承诺：话说得越庄严，越要问有没有路径、成本和救济；不能兑现的权利会消耗信任。

权力竞争最容易把人逼到只问输赢、不问手段。李敖批评这种抢夺姿态，提醒人即使在政治、职场或人际斗争中，也要知道哪些便宜不该占；没有底线的胜利，会让胜者也变小。

李敖谈抗议时看重效果，也警惕运动被政党、镜头和热闹劫持。人生中的反抗若只剩场面，就可能让真正诉求被表演盖住；行动可以有创意，但必须始终问一句：它是在解决问题，还是在喂养某些人的舞台感。

给别人写公开建议、道德劝告或政治忠告时，最怕自己的过去正好反着来。李敖提醒人，劝告不是不能给，而是要先承认自己曾怎样做；没有自我清账的教训，很容易变成漂亮的伪善。
