李敖书序集续集 人生智慧

条目数：43

【SXJX001】自由以人的处境为准
来源章节：001.《李敖语录香港版》感言.txt / 《李敖语录香港版》感言
主题标签：自由;人权;归属
判断一个地方可不可亲，不要只看它的旗号和血统，而要看它能不能让人说话、写作、求公道。自由与人权不是装饰性的词，它们常常比名义上的归属更能说明一个人实际站在哪里。

【SXJX002】制度的高低要看下限
来源章节：001.《李敖语录香港版》感言.txt / 《李敖语录香港版》感言
主题标签：制度;异议;底线
制度不只看它口号多响，也要看它在最不喜欢的人面前能不能留下基本空间。一个社会若连异议者的书、话和冤屈都容不下，再高的名义也会露出低标准。

【SXJX003】真相被封锁时要寻找通道
来源章节：001.《李敖语录香港版》感言.txt / 《李敖语录香港版》感言
主题标签：言论;真相;通道
一个地方不许你说，真相就要设法从别处说出去。人在压迫中不能只等环境开恩，还要寻找能保存声音、保存证据、保存读者的通道；通道本身也是抵抗的一部分。

【SXJX004】独立要有细密头脑
来源章节：003.《求是报合订本》广告词.txt / 《求是报合订本》广告词
主题标签：独立;判断;思想
独立不是跟着相反方向盲冲，而是在没有党派号令、没有教条拘束时，仍能客观、开明、细密地判断事情。真正的自由头脑，既不做群众动作的零件，也不做反向口号的囚徒。

【SXJX005】黑暗时代最怕知识人只求自保
来源章节：004.《国民党查禁李敖著作》序.txt / 《国民党查禁李敖著作》序
主题标签：知识人;勇气;时代
高压造成的冰河期并不只靠权力维持，也靠知识人把恐惧合理化、把沉默说成聪明。一个读书人若只求保全首领和身段，时代就会少一盏灯，多一层冷。

【SXJX006】禁令不能替你决定是否继续写
来源章节：004.《国民党查禁李敖著作》序.txt / 《国民党查禁李敖著作》序
主题标签：坚持;写作;封杀
坐牢、查禁、封杀都能改变作品的流通，却不该改变一个人继续下笔的秩序。真正的坚持不是喊得多响，而是在跌倒之后仍有常态：照写、照记、照把账留下来。

【SXJX007】为不平做耐心记录
来源章节：004.《国民党查禁李敖著作》序.txt / 《国民党查禁李敖著作》序
主题标签：记录;公道;证据
替冤案和禁书做表，看似琐碎，却是在替公道保存索引。很多不平被遗忘，不是因为它不存在，而是因为没人把日期、名目和证据一条条钉住。耐心也是一种反抗。

【SXJX008】多方证据免于被各派叙事骗
来源章节：005.《二二八研究》广告词.txt / 《二二八研究》广告词
主题标签：历史;证据;判断
面对复杂历史，最要紧的是眼观四面、无徵不信：官方材料、反对者叙述、受害者证词、旁观舆论，都要放到一起看。只听一边，很容易把别人的宣传当成自己的判断。

【SXJX009】取舍比堆积更见眼力
来源章节：006.《李敖作品精选集》总序.txt / 《李敖作品精选集》总序
主题标签：取舍;眼力;写作
写出许多文章是一种能力，从中选出真正能代表精神和水准的篇章，又是另一种能力。取舍难在要对自己下手，舍得割爱，才有精选；不舍得割，只有堆积。

【SXJX010】厌恶政治不等于任它摆布
来源章节：007.《洗你的脑，掐他脖子》自序.txt / 《洗你的脑，掐他脖子》自序
主题标签：政治;反抗;清醒
讨厌政治的人，若任由政治丑态支配生活，最后仍被政治收编。可以不爱权力游戏，但不能放弃对它的判断和反击；在必要时站出来，是为了不让自己被别人洗脑。

【SXJX011】人像与历史互相铸造
来源章节：008.简介两种.txt / 简介两种
主题标签：人物;历史;时代
看一个人物，不该只把他当成孤立传记；人像里面有历史，历史里面也有人像。时代塑造人的姿势，人又反过来改变时代的纹路，读人若不读时代，就会读薄。

【SXJX012】别在乱捧乱骂中丢掉真面目
来源章节：008.简介两种.txt / 简介两种
主题标签：人物;判断;证据
一个人被捧得太高或骂得太低，真面目都会被遮住。判断人物要穿过掌声和唾沫，看他的处境、行为、限制和证据；只有摆脱情绪队形，才看得到人本身。

【SXJX013】深刻思想要能讲清楚
来源章节：008.简介两种.txt / 简介两种
主题标签：思想;表达;学问
学问不是把话说得玄奥，让听者以为自己不配懂。真正有力量的思想，应该有广度，也能清清楚楚地打开人心；术语若只用来遮羞，反而暴露思想的贫乏。

【SXJX014】写人物靠证据，不靠丑化
来源章节：008.简介两种.txt / 简介两种
主题标签：人物;证据;写作
写人物不是先爱先恨，再去找材料配合。可靠的笔法，是把别人不知道的证据翻出来，让事实自己决定褒贬；有真凭实据时，不必加油添醋，也足够有力。

【SXJX015】朋友归朋友，是非归是非
来源章节：010.《黄旗梦碎》序.txt / 《黄旗梦碎》序
主题标签：朋友;是非;原则
真正的朋友关系，经不起用是非来交换。朋友有错，不能因为情分就取消判断；反过来，愿意把话说清楚的人，才可能是长期的朋友。避谈是非的友谊，往往只是互相纵容。

【SXJX016】公共证据不该被垄断
来源章节：010.《黄旗梦碎》序.txt / 《黄旗梦碎》序
主题标签：证据;公共利益;公开
牵涉公共利益的资料，不是某个人拿来独占风头的私产。证据若能帮助社会看清特权和弊端，就该开放给更多人使用、核对和追问；资料越公开，公道越有机会。

【SXJX017】问题要在还来得及的时候处理
来源章节：010.《黄旗梦碎》序.txt / 《黄旗梦碎》序
主题标签：时机;公共问题;行动
知道一件事会坏掉，却等到结局已定才说，常常只剩姿态，没有效果。公共问题尤其如此：警报要在还来得及改变时响起，迟到的正确，有时只是事后的漂亮话。

【SXJX018】失败要留下可用的教训
来源章节：010.《黄旗梦碎》序.txt / 《黄旗梦碎》序
主题标签：失败;反省;经验
失败若只被拿来怨人或遮羞，就白失败了。把失败过程记录下来，找出关键错误、性格限制和制度漏洞，才能把一场败局转成后来者的路标。

【SXJX019】不要提出自己付不起代价的路线
来源章节：012.《文茜半生缘》序.txt / 《文茜半生缘》序
主题标签：理想;代价;诚实
谈理想路线时，最容易慷他人之慨。一个人若要求别人立刻牺牲工作、家庭和安全，至少要先问自己能不能付同样代价。付不起的方案，不该轻易包装成勇敢。

【SXJX020】权力会放大人的暗处
来源章节：012.《文茜半生缘》序.txt / 《文茜半生缘》序
主题标签：权力;自省;人性
政治不是单纯的舞台，也像照妖镜，会把人心里贪、怕、妒、狠的一面放大。想进入权力场，就先承认自己也有暗处；不肯看见自己的魔鬼，最容易被权力牵着走。

【SXJX021】对恶规则说不，是在救自己
来源章节：012.《文茜半生缘》序.txt / 《文茜半生缘》序
主题标签：规则;人格;拒绝
有些比赛要求你不断退让、不断妥协，最后赢了位置却输了自己。遇到这种规则，拒绝不是失败，而是把自己从坏游戏里救出来；保住人格，有时比取得席位更重要。

【SXJX022】不把自由交给权力交换
来源章节：012.《文茜半生缘》序.txt / 《文茜半生缘》序
主题标签：自由;权力;选择
权力可以给人舞台，也会要人交出自由作为代价。最清醒的选择，是分得清自己真正要什么：宁可把权力留给别人，也不要让自由被权力包装后赎走。

【SXJX023】终极关怀不要交给政党
来源章节：012.《文茜半生缘》序.txt / 《文茜半生缘》序
主题标签：政党;理想;忠诚
政党只是手段，不该成为情感和理想的终点。一个人若真正关心土地、社会或人民，就要把目标放在更大的对象上，而不是把所有忠诚都押给会算计、会变形的组织。

【SXJX024】口号要跟着问题更新
来源章节：012.《文茜半生缘》序.txt / 《文茜半生缘》序
主题标签：口号;问题意识;现实
曾经锋利的口号，过了时也会变成旧壳。社会问题往前走，人的思想也要往前问：下一步是什么、代价是什么、现实如何承接。只重复旧口号，容易把清醒变成习惯。

【SXJX025】理想也要回答现实难题
来源章节：012.《文茜半生缘》序.txt / 《文茜半生缘》序
主题标签：理想;现实;责任
理想不因现实困难而取消，但也不能靠激情绕过现实。即使目标有一天成功，仍要面对邻近强权、经济条件和人心分裂等问题；敢梦是一回事，能收拾梦后的世界又是一回事。

【SXJX026】自悔少作是一种进步
来源章节：014.复瓿书成空自苦，击辕歌罢遣谁听.txt / 复瓿书成空自苦，击辕歌罢遣谁听
主题标签：自省;成长;写作
回头看早年的作品觉得不满意，不必全当羞耻；这常是眼力长出来的证明。怕的不是年轻时写得浅，而是多年以后仍护短，把昨天的自己当成不可批评的偶像。

【SXJX027】旧作可当史料，不必当代表作
来源章节：014.复瓿书成空自苦，击辕歌罢遣谁听.txt / 复瓿书成空自苦，击辕歌罢遣谁听
主题标签：旧作;史料;自我评价
早期文字即使不成熟，也可能保存一代人的气味、问题和限制。对旧作最好的态度，不一定是销毁或神化，而是把它放回历史位置：让它作证，不必让它撑门面。

【SXJX028】好写作能在旧与新之间翻出自己
来源章节：015.序凉如水.txt / 序凉如水
主题标签：写作;创新;自知
写作不只是把新鲜题材堆上纸面，也可以把旧事说出新意，或用旧形式安放新经验。真正值得写的东西，常常在这种转换中逼出作者对自己的认识。

【SXJX029】刑求之下口供最不可靠
来源章节：016.《调查局黑牢345天》序.txt / 《调查局黑牢345天》序
主题标签：刑求;真相;司法
刑求不是通向真相的捷径，而是制造口供的机器。人在痛苦和恐惧下会说出审问者想听的话，案件也会被一步步绣成看似完整的故事；越是来得容易的供词，越要怀疑。

【SXJX030】抽象控诉要落到血肉实例
来源章节：016.《调查局黑牢345天》序.txt / 《调查局黑牢345天》序
主题标签：证词;历史;制度
说制度残酷，若只有抽象名词，常让后来人感受不到重量。真正能补历史空缺的，是具体的人、天数、伤口、逼供细节和命运转折；血肉实例能把空洞控诉变成不可回避的证词。

【SXJX031】迷信常被用来遮盖无能
来源章节：017.《好命操作手册》序.txt / 《好命操作手册》序
主题标签：迷信;判断;责任
把命运交给算命和补命，常常会替人的懒惰、统治者的无能和社会的焦虑找到借口。越是混乱的时代，越要警惕这种安慰剂；它让人以为解释了命运，其实放弃了判断。

【SXJX032】人生不只是活过一辈子
来源章节：018.陆晋德译《飞鸟集》序.txt / 陆晋德译《飞鸟集》序
主题标签：人生;审美;时间
活着不等于把日子从头拖到尾。人生由许多片刻组成，若没有爱、审美、醒悟和超出实用的追问，时间就只剩消耗。让片刻有光，才算不只是活了一辈子。

【SXJX033】阅读的乐趣在理解与想象之间
来源章节：018.陆晋德译《飞鸟集》序.txt / 陆晋德译《飞鸟集》序
主题标签：阅读;想象;理解
读诗或读好书，不必把每一句都钉死成唯一解释。作者有来意，读者也有想象；真正的阅读，是在理解作者和唤醒自己之间来回走动，在半明半悟里得到滋味。

【SXJX034】敌友可因人格重新划线
来源章节：019.国家将兴，必有祯祥.txt / 国家将兴，必有祯祥
主题标签：人格;敌友;原则
敌友并不是永远按阵营固定。一个人即使曾站在对面，只要在关键处不巴结权力、不出卖人格、肯为事实付代价，就值得重新评价。真正的分界线，常在品格而不在旧标签。

【SXJX035】大方向相同不必强求全同
来源章节：019.国家将兴，必有祯祥.txt / 国家将兴，必有祯祥
主题标签：合作;差异;求同存异
人与人合作，不必把所有历史细节、性格好恶和表达方式都磨成一样。只要大方向相同，就可以求同存异、互通有无；成熟的同道关系，容得下差异。

【SXJX036】政治会腐化，文字能留下正气
来源章节：019.国家将兴，必有祯祥.txt / 国家将兴，必有祯祥
主题标签：文字;政治;正气
权力场容易把人拖向算计，但文字若诚实，仍能替时代留下另一种气息。政治人物可能被历史埋掉，思想和文章却可能继续作证；把正气写下来，是给未来留一条线索。

【SXJX037】地位上升不等于自由增加
来源章节：020.我看《宰相刘罗锅》.txt / 我看《宰相刘罗锅》
主题标签：自由;地位;表达
一个人或行业获得更高地位，不必然获得更大自由；有时越被看见，越被管理。判断进步不能只看身份抬高，还要看能不能说真话、开玩笑、讽刺权力。

【SXJX038】受限环境里讽刺是出口
来源章节：020.我看《宰相刘罗锅》.txt / 我看《宰相刘罗锅》
主题标签：讽刺;表达;限制
当直说被堵住，民间常会借古说今、借笑话说痛处。讽刺不只是娱乐，也是一种绕过禁忌的表达技术；它让人们在不能公开怒骂时，仍保留一点清醒和痛快。

【SXJX039】政治正确会压过事实正确
来源章节：021.是真的也要否认.txt / 是真的也要否认
主题标签：事实;政治正确;真相
有些真相并非无人知道，而是因“承认了会破坏大局”而被集体否认。政治正确一旦凌驾事实正确，人们就会先问能不能说，再问是不是真的。

【SXJX040】离谱不等于不真
来源章节：021.是真的也要否认.txt / 是真的也要否认
主题标签：证据;判断;历史
听起来不合常理的事，不必立刻当成谣言，也不该轻信。历史里许多“不可能”，后来靠档案、旁证和当事人缝隙被证实；判断离奇之事，关键不是顺耳，而是证据。

【SXJX041】为事实得罪强者
来源章节：021.是真的也要否认.txt / 是真的也要否认
主题标签：事实;勇气;权力
追求事实正确，常常要冒犯拥有解释权的人。若只在安全范围内说真话，真话就会被权力划界；敢为历史和事实得罪强者，是读书人最难也最必要的勇气。

【SXJX042】幕后搜集考订也是功夫
来源章节：022.《胡适选集》五十年.txt / 《胡适选集》五十年
主题标签：整理;学问;编选
整理、搜集、校订、编选，看起来不如原创显眼，却同样考验功力。一个人肯匿名把散落文字找齐、比勘、排出脉络，是在替知识做地基；地基不露脸，却撑住大楼。

【SXJX043】真懂一个人要找到他遗漏的文字
来源章节：022.《胡适选集》五十年.txt / 《胡适选集》五十年
主题标签：阅读;搜集;人物研究
理解一位思想者，不能只读最流行、最自我标榜的文本，还要追索他散落在报刊、书信和旧刊物里的遗漏之作。越能补齐边角，越能接近一个人的全貌。
